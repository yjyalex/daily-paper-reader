## 摘要
信用分配是强化学习（RL）中的核心挑战。经典的演员-评论家（actor-critic）方法通过基于学习到的价值函数进行细粒度的优势估计来应对这一挑战。然而，在现代大语言模型（LLM）强化学习中，人们往往避免使用学习到的价值模型，因为传统的判别式评论家难以进行可靠的训练。我们重新审视了价值建模，并认为这种困难部分归因于表达能力的局限性。特别是，表示复杂度理论表明，在现有价值模型所采用的一次性预测范式下，价值函数可能难以近似；我们的缩放实验也表明，此类评论家的性能并不会随着规模的扩大而可靠地提升。受此启发，我们提出了生成式演员-评论家（GenAC），它将一次性标量价值预测替换为生成式评论家，该评论家在生成价值估计之前会进行思维链（chain-of-thought）推理。我们还引入了上下文调节（In-Context Conditioning），这有助于评论家在整个训练过程中与当前的演员保持校准。GenAC 提高了价值近似、排序可靠性和分布外泛化能力，这些提升转化为比有价值模型和无价值模型基准更强的下游强化学习性能。总之，我们的结果表明，更强大的价值建模是改进大语言模型强化学习中信用分配的一个极具前景的方向。

## Abstract
Credit assignment is a central challenge in reinforcement learning (RL). Classical actor-critic methods address this challenge through fine-grained advantage estimation based on a learned value function. However, learned value models are often avoided in modern large language model (LLM) RL because conventional discriminative critics are difficult to train reliably. We revisit value modeling and argue that this difficulty is partly due to limited expressiveness. In particular, representation complexity theory suggests that value functions can be hard to approximate under the one-shot prediction paradigm used by existing value models, and our scaling experiments show that such critics do not improve reliably with scale. Motivated by this observation, we propose Generative Actor-Critic (GenAC), which replaces one-shot scalar value prediction with a generative critic that performs chain-of-thought reasoning before producing a value estimate. We further introduce In-Context Conditioning, which helps the critic remain calibrated to the current actor throughout training. GenAC improves value approximation, ranking reliability, and out-of-distribution generalization, and these gains translate into stronger downstream RL performance than both value-based and value-free baselines. Overall, our results suggest that stronger value modeling is a promising direction for improving credit assignment in LLM reinforcement learning.

    *   传统的 PPO 算法依赖判别式评论家（Discriminative Critic）预测标量价值，但这种模型在 LLM 规模下极难训练且极不稳定。
    *   近期趋势（如 GRPO、RLOO）倾向于弃用价值模型，改用简单的轨迹级平均奖励，但这牺牲了细粒度的反馈，限制了模型处理复杂长程推理的能力。

### 2. 方法论：核心思想与关键技术
*   **核心思想**：将价值预测从“标量回归”转变为“生成式推理”。让评论家在给出分值前，先进行思维链（CoT）推理，分析当前状态的优劣。
*   **关键技术细节**：
    *   **生成式评论家（Generative Critic）**：保留预训练模型的语言头，提示模型生成推理过程，最后输出一个 0-10 的整数分值，再归一化为 [0, 1] 的价值估计。
    *   **上下文调节（In-Context Conditioning, ICC）**：价值函数必须随策略（Actor）的变化而变化。GenAC 在提示词中加入当前策略的描述（如模型大小、训练实时胜率），使评论家能“感知”当前演员的水平，从而实现更精准的校准。
*   **算法流程（GenAC）**：
    1.  **SFT 阶段**：利用 GPT-5 生成高质量推理轨迹，让评论家学会评价的“格式”。
    2.  **RL 预训练阶段**：冻结演员，用 REINFORCE 算法训练评论家，目标是最小化预测价值与实际回报的均方误差（MSE）。
    3.  **联合训练阶段**：演员（PPO）与评论家（REINFORCE）交替更新，确保价值模型紧跟策略演进。

*   **实验规模**：

### 6. 主要结论与发现
1.  **表达能力是瓶颈**：判别式评论家无法通过单纯增加参数量来提升准确度，而生成式评论家随规模扩大表现出明显的 Scaling Law。
2.  **性能领先**：GenAC 在所有数学基准测试中均优于 GRPO 和 VC-PPO，且样本效率更高（达到相同精度所需的训练步数更少）。
3.  **排序更可靠**：生成式评论家在对不同候选答案进行排序时，准确率显著高于传统方法。
4.  **泛化能力强**：在面对未见过的难题或跨领域任务（GPQA）时，GenAC 的价值估计误差远低于基线。

### 7. 优点
*   **理论支撑扎实**：从表示复杂度理论出发，解释了为什么传统 Critic 会失效，为后续研究指明了方向。
*   **可解释性**：评论家输出的 CoT 推理过程让信用分配变得透明，开发者可以直观看到模型为何给某个中间步骤打高分或低分。
*   **即插即用**：ICC 机制提供了一种轻量级的方法，让通用模型快速适配特定策略的评估。

### 8. 不足与局限
*   **计算开销**：虽然比采样法便宜，但 2.1 倍的开销在超大规模模型训练中仍是不小的负担。
*   **任务局限**：目前主要在数学推理（逻辑性强、有明确答案）上验证，在创意写作或开放式对话等主观性更强的任务中，生成式评论家的表现尚待验证。
*   **流程复杂**：相比 GRPO 的简洁，GenAC 需要经历 SFT、预训练和联合训练多个阶段，增加了工程实现的复杂度。
