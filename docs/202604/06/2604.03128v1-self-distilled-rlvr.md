## 摘要
同策略蒸馏（OPD）已成为大语言模型（LLM）社区中一种流行的训练范式。该范式选择一个较大的模型作为教师，为每个采样轨迹提供密集、细粒度的信号，这与具有可验证奖励的强化学习（RLVR）形成对比，后者仅从环境中的可验证结果中获取稀疏信号。最近，社区探索了同策略自蒸馏（OPSD），其中同一个模型同时充当教师和学生，教师接收额外的特权信息（如参考答案）以实现自我进化。本文证明，仅源自特权教师的学习信号会导致严重的信息泄漏和不稳定的长期训练。因此，我们确定了自蒸馏的最佳定位，并提出了 RLSD（结合自蒸馏的 RLVR）。具体而言，我们利用自蒸馏获取 Token 级的策略差异，以确定细粒度的更新幅度，同时继续使用 RLVR 从环境反馈（例如回答的正确性）中获取可靠的更新方向。这使得 RLSD 能够同时利用 RLVR 和 OPSD 的优势，实现更高的收敛上限和卓越的训练稳定性。

## Abstract
On-policy distillation (OPD) has become a popular training paradigm in the LLM community. This paradigm selects a larger model as the teacher to provide dense, fine-grained signals for each sampled trajectory, in contrast to reinforcement learning with verifiable rewards (RLVR), which only obtains sparse signals from verifiable outcomes in the environment. Recently, the community has explored on-policy self-distillation (OPSD), where the same model serves as both teacher and student, with the teacher receiving additional privileged information such as reference answers to enable self-evolution. This paper demonstrates that learning signals solely derived from the privileged teacher result in severe information leakage and unstable long-term training. Accordingly, we identify the optimal niche for self-distillation and propose \textbf{RLSD} (\textbf{RL}VR with \textbf{S}elf-\textbf{D}istillation). Specifically, we leverage self-distillation to obtain token-level policy differences for determining fine-grained update magnitudes, while continuing to use RLVR to derive reliable update directions from environmental feedback (e.g., response correctness). This enables RLSD to simultaneously harness the strengths of both RLVR and OPSD, achieving a higher convergence ceiling and superior training stability.

# 论文总结：Self-Distilled RLVR (RLSD)

### 1. 核心问题与研究背景
*   **研究动机**：在训练大语言模型（LLM）的推理能力时，**RLVR（具有可验证奖励的强化学习）**（如 GRPO）虽然可靠，但由于奖励信号仅在序列末尾提供（稀疏奖励），导致收敛缓慢且 Token 级信用分配困难。
*   **现有方案的缺陷**：
    *   **OPD（在线蒸馏）**：需要一个更强大的外部教师模型，计算开销大且受限于词表一致性。
    *   **OPSD（在线自蒸馏）**：让模型自身在获取“特权信息”（如参考答案）后充当教师。虽然高效，但会导致**特权信息泄露**（模型在推理时幻觉出不存在的参考答案）和训练后期性能崩塌。
*   **核心问题**：如何既能利用自蒸馏提供的密集、细粒度信号，又能避免信息泄露并保持训练稳定性？

### 2. 方法论：RLSD (RLVR with Self-Distillation)
*   **核心思想**：将自蒸馏信号从“生成目标”转变为“更新幅度评估器”。**环境奖励决定更新方向，自蒸馏信号决定更新幅度。**
*   **关键技术细节**：
    1.  **特权信息增益 ($\Delta_t$)**：计算 Token 在“教师模式”（输入+特权信息）和“学生模式”（仅输入）下的 Log 概率差。
    2.  **方向感知证据调优 ($w_t$)**：定义权重 $w_t = \exp(\text{sign}(A) \cdot \Delta_t)$。
        *   如果序列奖励 $A > 0$（正确）：教师支持的 Token 获得更大的正向权重。
        *   如果序列奖励 $A < 0$（错误）：教师反对的 Token 承担更大的惩罚权重。
    3.  **裁剪信用分配**：引入类似 PPO 的裁剪机制 $\text{clip}(w_t, 1-\epsilon_w, 1+\epsilon_w)$，限制单个 Token 对梯度的影响，确保训练稳定性。
*   **算法流程**：
    *   模型生成多个 Rollout。
    *   通过环境验证器获取正确性奖励（序列级）。
    *   进行一次额外的教师模式前向传播，计算 Token 级的证据比（Evidence Ratio）。
    *   将该比例作为权重作用于 GRPO 的策略梯度，更新模型参数。

### 6. 主要结论与发现
*   **OPSD 失败的根源**：由于信息不对称，自蒸馏的分布匹配目标是“病态”的（Ill-posed），会导致模型参数强制编码输入与特权信息的相关性，从而引发泄露。
*   **RLSD 的优越性**：RLSD 在所有基准上均显著优于 GRPO 和 OPSD。在 200 步时的表现甚至超过了训练 400 步的 GRPO，证明了其**极高的收敛效率**。
*   **稳定性**：RLSD 成功解决了自蒸馏后期性能下降的问题，保持了与 GRPO 相当的训练稳定性，同时拥有更高的性能上限。

### 7. 优点
*   **理论深刻**：深入剖析了自蒸馏在强化学习中的内在矛盾（不可能三角：稳定性、持续改进、无泄露）。
*   **设计精巧**：通过解耦方向和幅度，将原本有害的泄露信号转化为有益的信用分配信号。
*   **低成本**：不需要额外的教师模型，仅增加一次前向传播开销，非常适合大规模工业化训练。
