## 摘要
过程监督已成为增强大语言模型（LLM）推理能力的一种极具前景的方法，然而现有方法无法区分有意义的进展与单纯的冗余，导致推理能力受限且 Token 效率低下的问题仍未解决。为了应对这一挑战，我们提出了 SHAPE（通过势能估计实现的阶段感知层级优势），这是一个将推理形式化为经验可解性状态空间中轨迹的框架。SHAPE 引入了一种层级信用分配机制：在片段级，它采用阶段感知优势函数，优先考虑低势能状态下的高效突破；在 Token 级，它利用熵驱动的重新分配来强化执行信号。在三个基座模型和五个基准测试上进行的广泛数学推理实验表明，SHAPE 在减少 30% Token 消耗的同时，实现了平均 3% 的准确率提升。

## Abstract
Process supervision has emerged as a promising approach for enhancing LLM reasoning, yet existing methods fail to distinguish meaningful progress from mere verbosity, leading to limited reasoning capabilities and unresolved token inefficiency. To address this, we propose Stage-aware Hierarchical Advantage via Potential Estimation (SHAPE), a framework that formalizes reasoning as a trajectory through a state space of empirical solvability. SHAPE introduces a hierarchical credit assignment mechanism: at the segment level, it employs a stage-aware advantage function to prioritize efficient breakthroughs in low-potential states; at the token level, it utilizes entropy-driven redistribution to sharpen execution signals. Extensive experiments in math reasoning across three base models and five benchmarks demonstrate that SHAPE achieves an average accuracy gain of 3% with 30% reduced token consumption.

    2.  **信用分配不均**：现有的过程奖励往往忽略了推理阶段的难度差异，且无法精准识别推理路径中的关键决策点。

### 2. 方法论
SHAPE 将推理过程建模为状态空间中的轨迹，核心思想是**层级化信用分配**：
*   **推理势能（Reasoning Potential, $\Phi$）**：定义为从当前状态出发，通过多次采样（Rollouts）能够得到正确答案的经验概率。
*   **片段级（Segment-level）优势函数**：
    *   **动态折扣因子（$\gamma_k$）**：引入与片段长度 $L_k$ 相关的折扣因子。片段越长，折扣越大，从而产生“推理税”，惩罚冗余。
    *   **阶段感知奖励**：公式为 $A_k = R_{outcome} + \alpha \cdot (\gamma_k \Phi(s_{k+1}) - \Phi(s_k))$。该设计确保了从低势能状态（困难阶段）取得的突破比在高势能状态下的微调获得更高奖励。
*   **Token 级（Token-level）信用重分布**：
    *   利用**Token 熵（Entropy）**识别关键决策点。高熵 Token 通常代表逻辑分支点，通过 Z-score 标准化将片段奖励向这些关键 Token 倾斜，强化执行信号。
*   **自适应分割（EBS）**：放弃固定的换行符分割，改用基于熵的采样点来识别语义转折点，使势能估计更符合逻辑结构。

### 8. 不足与局限
*   **领域局限**：目前主要集中在具有确定性验证（Deterministic Verifiability）的数学任务上。对于创意写作、开放式对话等难以量化“势能”的领域，其适用性尚待验证。
