## 摘要
当自主智能体观察共享环境的不同领域时，它们交换的每个信号都混合了相关和不相关的维度。目前尚无机制能让接收者评估应吸收哪些维度。我们引入了符号-向量注意力融合（SVAF），这是用于集体智能的两级耦合引擎中的内容评估部分。SVAF 将每个智能体间的信号分解为 7 个有类型的语义字段，通过学习到的融合门对每个字段进行评估，并产生一个“重混（remix）”——即来自两个领域交集的新知识。一种带通模型产生四种结果（冗余、对齐、防御、拒绝），同时解决了选择性和冗余问题。融合门独立地发现了一个跨领域的相关性层级：在准确率达到平台期之前，情绪（mood）在第 1 个 epoch 就成为了权重最高的字段——这与独立的机制性证据相一致，即大语言模型（LLM）的情感表示在结构上嵌入于效价-唤醒（valence-arousal）轴。SVAF 构成了网格记忆协议（MMP）的第 4 层；耦合引擎的另一部分是位于第 6 层的每个智能体特有的闭式连续时间（CfC）神经网络，其学习到的每个神经元的时间常数（tau）创造了集体智能涌现的时间动力学：快速神经元在数秒内同步智能体间的情感，而慢速神经元则无限期地保留领域专业知识。SVAF 决定了什么进入每个智能体的认知状态；CfC 决定了该状态如何演化。在来自 273 个叙事场景的 23.7 万个样本上进行训练，SVAF 实现了 78.7% 的三分类准确率。我们在跨 macOS、iOS 和 Web 的 7 个节点的实时部署中，验证了完整的网格认知闭环——从逐字段评估到重混、CfC 状态演化、tau 调制的同伴融合以及自主行动。

## Abstract
When autonomous agents observe different domains of a shared environment, each signal they exchange mixes relevant and irrelevant dimensions. No existing mechanism lets the receiver evaluate which dimensions to absorb. We introduce Symbolic-Vector Attention Fusion (SVAF), the content-evaluation half of a two-level coupling engine for collective intelligence. SVAF decomposes each inter-agent signal into 7 typed semantic fields, evaluates each through a learned fusion gate, and produces a remix -- new knowledge from the intersection of two domains. A band-pass model yields four outcomes (redundant, aligned, guarded, rejected), solving both selectivity and redundancy. The fusion gate independently discovers a cross-domain relevance hierarchy: mood emerges as the highest-weight field by epoch 1, before accuracy plateaus -- consistent with independent mechanistic evidence that LLM emotion representations are structurally embedded along valence-arousal axes. SVAF forms Layer 4 of the Mesh Memory Protocol (MMP); the other half of the coupling engine is a per-agent Closed-form Continuous-time (CfC) neural network at Layer 6, whose learned per-neuron time constants (tau) create the temporal dynamics from which collective intelligence emerges: fast neurons synchronise affect across agents in seconds, while slow neurons preserve domain expertise indefinitely. SVAF determines what enters each agent's cognitive state; CfC determines how that state evolves. Trained on 237K samples from 273 narrative scenarios, SVAF achieves 78.7% three-class accuracy. We verify the complete mesh cognition loop -- from per-field evaluation through remix, CfC state evolution, tau-modulated peer blending, and autonomous action -- in a live deployment with 7 nodes across macOS, iOS, and web.

#### 1. 核心问题与整体含义
在多智能体系统（MAS）中，不同领域的自主智能体（如编程助手、健身教练、音乐推荐）共同观察环境。当它们交换信息时，信号中往往混合了对接收者有用和无用的维度。现有的机制（如单一向量的余弦相似度）无法让接收者精确筛选应吸收哪些信息，也难以区分信息的冗余性与新颖性。
**研究动机**：解决跨领域信号评估问题，即如何让智能体自主决定吸收外部信号的哪些部分，并将其与本地记忆“重混（Remix）”以产生新知识，同时保持各自的领域专业性。

#### 2. 方法论
论文提出了 **SVAF (Symbolic-Vector Attention Fusion)**，它是网格记忆协议（MMP）的核心组件。
*   **CAT7 模式**：将每个信号分解为 7 个语义字段：核心关注（Focus）、问题（Issue）、意图（Intent）、动机（Motivation）、承诺（Commitment）、视角（Perspective）和情绪（Mood）。
*   **两级耦合架构**：
    *   **同伴级（Peer-level）**：评估智能体间的认知距离。
    *   **内容级（SVAF）**：评估具体信号的字段相关性。
*   **SVAF 流水线**：
    1.  **编码**：使用共享骨干网络和逐字段投影头生成符号-向量双重表示。

    3.  **评估与融合**：利用学习到的**融合门（Fusion Gate）**根据接收者的领域权重（$\alpha_f$）和时间新鲜度（$\tau$）决定各字段的吸收比例。
    4.  **重混（Remix）**：产生新知识并记录谱系（Lineage），而非简单复制。
*   **时间动力学**：结合 **CfC（闭式连续时间）神经网络**，利用神经元的时间常数（$\tau$）实现快速情感同步（快神经元）与长期专业知识保留（慢神经元）。

*   **情感优先**：融合门自动发现**情绪（Mood）**是跨领域相关性最高的字段，在训练初期就获得了最高权重，这与 LLM 内部情感表示的机制性证据一致。
*   **冗余检测**：通过带通模型成功区分了语义重复（冗余）和真正的新知识。
*   **冷启动能力**：即使在同伴耦合度极低的情况下，SVAF 也能通过内容评估实现有效的知识吸收。

#### 7. 优点
*   **结构化解耦**：CAT7 模式为异构智能体协作提供了统一的“语言”，解决了维度混杂问题。
*   **时间尺度分层**：利用 CfC 的 $\tau$ 参数巧妙地平衡了集体意识（情感同步）与个体主权（专业知识）。
*   **协议层优化**：将上下文工程（Context Engineering）下沉到协议层，减轻了应用层 LLM 的推理负担。
*   **可审计性**：通过重混谱系和门控值，使智能体间的决策过程可追溯。

#### 8. 不足与局限
*   **合成数据偏差**：训练数据主要由 LLM 生成的叙事场景组成，可能无法完全覆盖真实人类交互的所有复杂性。

*   **单用户限制**：目前的实时部署验证主要集中在单用户多设备场景，多用户间的复杂博弈和共识机制尚待进一步研究。
