## 摘要
大语言模型（LLMs）日益依赖智能体能力——迭代检索、工具使用和决策——以克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了弥补这一差距，我们引入了智能体图学习（Agentic Graph Learning, AGL），这是一种将图学习重新定义为拓扑感知导航与基于 LLM 推理的交替过程的范式。具体而言，我们提出了 AgentGL，这是首个用于 AGL 的强化学习（RL）驱动框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强力的 GraphLLMs 和 GraphRAG 基准模型，在节点分类和链路预测中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已在 https://github.com/sunyuanfu/AgentGL 公开。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

#### 1. 核心问题与整体含义（研究动机和背景）
*   **研究动机**：尽管大语言模型（LLMs）在处理非结构化文本方面表现出色，但在处理具有复杂拓扑依赖关系的**文本属性图（TAGs）**（如引用网络、社交平台、电商生态）时仍面临挑战。
*   **现有局限**：
    *   **GraphLLMs**：通常依赖静态的图上下文，缺乏推理时的自适应探索能力。
    *   **GraphRAG**：多基于从文本重建的知识图谱，而非原生拓扑结构，且构建成本高。
    *   **通用 Agent**：主要操作非结构化文本，无法感知图的结构化特征。
*   **核心问题**：如何构建一个能够自主导航图结构、动态获取证据并进行拓扑感知推理的智能体系统？

#### 2. 方法论：核心思想与技术细节
AgentGL 将图学习重新定义为一个由强化学习（RL）驱动的交互式决策过程。
*   **核心思想**：通过配备图原生搜索工具，让 LLM 在推理过程中自主决定“去哪里搜索”以及“何时停止”，实现拓扑感知与语义推理的交替。
*   **关键技术细节**：
    *   **图原生搜索工具集 (GNS Tools)**：包括 1-hop/2-hop 邻域搜索（局部）、结构显著性搜索（基于 PPR 分数，全局结构）和图稠密搜索（基于语义向量，全局语义）。
    *   **两阶段训练策略**：
        1.  **策略引导阶段 (Bootstrapping)**：使用奖励塑造（包括格式、准确性和工具覆盖率奖励）来培养智能体的工具使用能力。
        2.  **减少过度搜索阶段 (MSO)**：引入“搜索约束思维”（Search-Constrained Thinking），通过回顾性终止触发器和认知密度正则化，惩罚冗余搜索，鼓励深度推理。
    *   **图条件课程学习 (GCCL)**：利用图的拓扑先验（如节点度、同质性）和语义先验（余弦相似度）自动评估样本难度，实现从易到难的平滑训练。
    *   **优化算法**：支持 GRPO 和 REINFORCE++ 等无评论员（Critic-free）的策略优化算法。

#### 6. 主要结论与发现
*   **性能卓越**：AgentGL 在所有任务上均显著优于基准。在节点分类中最高提升 **17.5%**，在链路预测中最高提升 **28.4%**。

*   **效率平衡**：通过第二阶段训练，模型在保持高精度的同时，平均减少了约 **17.5% - 22%** 的工具调用次数。

#### 7. 优点（亮点）
*   **范式创新**：首次将图学习定义为 RL 驱动的智能体导航过程，统一了结构感知与语义推理。
*   **工具设计**：设计的 GNS 工具集兼顾了局部与全局、结构与语义，覆盖了图信息的全空间。
*   **训练稳定性**：引入 GCCL 课程学习，利用图固有属性量化难度，有效解决了长程策略学习难以收敛的问题。
*   **效率优化**：MSO 阶段的设计体现了“少即是多”的原则，避免了 LLM 常见的“过度搜索”和“信息过载”问题。

#### 8. 不足与局限
*   **模态限制**：目前仅支持文本属性图，尚未扩展到包含图像、视频等多模态属性的图结构。
*   **图密度敏感性**：在极稠密图上的扩展性和计算开销仍需进一步探索。
*   **训练依赖**：MSO 阶段的稳定性高度依赖于两个训练阶段之间的数据分配比例。
*   **推理成本**：尽管优化了搜索次数，但多轮交互式推理的延迟（Latency）仍高于传统的单次推理方法。
