## 摘要
大语言模型智能体已迅速从静态文本生成器演变为能够执行复杂自主工作流的动态系统。为了增强可靠性，越来越多地采用分配专门角色的多智能体框架，以实现自我反思和相互审计。虽然这种角色扮演有效地利用了领域专家知识，但我们发现它同时诱发了一种类似于人类的认知偏差，即行动者-观察者不对称性（AOA）。具体而言，作为行动者的智能体（在自我反思期间）倾向于将失败归因于外部因素，而观察者（在相互审计期间）则将同样的错误归因于内部缺陷。我们使用新的“模糊失败基准测试”（Ambiguous Failure Benchmark）对此进行了量化，结果显示，对于大多数模型，仅通过交换视角就会在超过 20% 的案例中触发 AOA 效应。为了抑制这种偏差，我们引入了 ReTAS（通过正题-反题-合题进行推理），这是一个通过辩证对齐训练的模型，旨在强制执行视角不变的推理。通过将辩证思维链与群体相对策略优化（GRPO）相结合，ReTAS 引导智能体将冲突的观点综合为客观的共识。实验表明，ReTAS 有效地缓解了归因不一致性，并显著提高了模糊场景下的故障解决率。

## Abstract
Large Language Model agents have rapidly evolved from static text generators into dynamic systems capable of executing complex autonomous workflows. To enhance reliability, multi-agent frameworks assigning specialized roles are increasingly adopted to enable self-reflection and mutual auditing. While such role-playing effectively leverages domain expert knowledge, we find it simultaneously induces a human-like cognitive bias known as Actor-Observer Asymmetry (AOA). Specifically, an agent acting as an actor (during self-reflection) tends to attribute failures to external factors, whereas an observer (during mutual auditing) attributes the same errors to internal faults. We quantify this using our new Ambiguous Failure Benchmark, which reveals that simply swapping perspectives triggers the AOA effect in over 20% of cases for most models. To tame this bias, we introduce ReTAS (Reasoning via Thesis-Antithesis-Synthesis), a model trained through dialectical alignment to enforce perspective-invariant reasoning. By integrating dialectical chain-of-thought with Group Relative Policy Optimization, ReTAS guides agents to synthesize conflicting viewpoints into an objective consensus. Experiments demonstrate that ReTAS effectively mitigates attribution inconsistency and significantly improves fault resolution rates in ambiguous scenarios.

*   **研究动机**：在多智能体框架中，通常会分配“执行者（行动者）”和“审计者（观察者）”角色。研究发现，当任务失败时，执行者倾向于将错误归因于外部环境（如指令模糊、服务器问题），而审计者则倾向于归因于执行者的内部缺陷（如逻辑错误、能力不足）。这种归因不一致会导致智能体间的对齐失效，降低协作的可靠性。
*   **背景**：虽然角色扮演能激发专家知识，但它也引入了角色偏见。现有的“自我反思”或“交叉审核”机制往往只是在强化这种偏见，而非消除它。

### 2. 核心方法论：ReTAS 框架
论文提出了 **ReTAS (Reasoning via Thesis-Antithesis-Synthesis)**，通过辩证对齐（Dialectical Alignment）来抑制归因偏差。
*   **核心思想**：借鉴费希特的辩证法，将推理过程分解为三个显式阶段：
    1.  **正题 (Thesis)**：生成符合角色设定的初始解释（体现角色偏见）。
    2.  **反题 (Antithesis)**：模拟对立视角或检查客观证据，寻找盲点。
    3.  **合题 (Synthesis)**：调和冲突观点，基于客观证据得出视角不变的结论。
*   **关键技术细节**：
    *   **数据生成**：利用强模型（如 GPT-5.1）生成包含“正-反-合”结构的推理轨迹。
    *   **训练流程**：
        *   **监督微调 (SFT)**：让模型学习 TAS 的结构化格式。
        *   **强化学习对齐 (GRPO)**：使用群体相对策略优化（Group Relative Policy Optimization），通过复合奖励函数进行训练。
    *   **奖励函数**：$R = \alpha R_{格式} + \beta R_{归因准确} + \gamma R_{答案正确}$。通过奖励归因的一致性和最终答案的准确性，迫使模型内化辩证推理能力。

### 6. 主要结论与发现
*   **AOA 普遍存在**：即使是最先进的模型（如 GPT-5.1），在视角切换时也会出现归因不一致，小模型尤为严重（Flip 率常超过 20%）。
*   **规模非万能**：单纯增加参数规模不能消除认知偏差，结构化的辩证推理训练更为有效。
*   **ReTAS 效果显著**：经过训练的 4B 模型在归因准确性和一致性上超过了许多 30B+ 规模的模型，并显著提升了下游任务的故障解决率。
*   **归因平衡**：ReTAS 能够打破“总是偏袒人类”或“总是自我防卫”的思维定式，实现基于证据的客观归因。
