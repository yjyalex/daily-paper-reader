## 摘要
基于大语言模型（LLM）的多智能体系统（MAS）在处理复杂任务方面展现出巨大潜力，但仍容易出现协作失败，如目标漂移、错误级联和行为失调。我们提出了显式特质推理（Explicit Trait Inference, ETI），这是一种基于心理学理论的改进协作的方法。ETI 使智能体能够从交互历史中推断并追踪合作伙伴在两个经典心理学维度——热情（如信任）和能力（如技能）——上的特征，并以此指导决策。我们在受控环境（经济博弈）中对 ETI 进行了评估，结果显示其将收益损失降低了 45-77%；在更现实、复杂的多智能体场景（MultiAgentBench）中，相较于思维链（CoT）基准，其性能提升了 3-29%（取决于具体场景和模型）。进一步的分析表明，性能增益与特质推理密切相关：ETI 生成的画像能够预测智能体的行为，且信息丰富的画像是性能提升的关键驱动力。这些结果表明，ETI 是一种在多样化多智能体环境中改进协作的轻量级且鲁棒的机制，并提供了首个系统性证据，证明 LLM 智能体能够：(i) 从交互历史中可靠地推断他人的特质；(ii) 利用对他人的结构化特质认知来优化协作。

## Abstract
LLM-based multi-agent systems (MAS) show promise on complex tasks but remain prone to coordination failures such as goal drift, error cascades, and misaligned behaviors. We propose Explicit Trait Inference (ETI), a psychologically grounded method for improving coordination. ETI enables agents to infer and track partner characteristics along two established psychological dimensions--warmth (e.g., trust) and competence (e.g., skill)--from interaction histories to guide decisions. We evaluate ETI in controlled settings (economic games), where it reduces payoff loss by 45-77%, and in more realistic, complex multi-agent settings (MultiAgentBench), where it improves performance by 3-29% depending on the scenario and model, relative to a CoT baseline. Additional analysis shows that gains are closely linked to trait inference: ETI profiles predict agents' actions, and informative profiles drive improvements. These results highlight ETI as a lightweight and robust mechanism for improving coordination in diverse multi-agent settings, and provide the first systematic evidence that LLM agents can (i) reliably infer others' traits from interaction histories and (ii) leverage structured awareness of others' traits for coordination.

*   **核心问题**：基于大语言模型（LLM）的多智能体系统（MAS）在处理复杂任务时，常因“协调失败”（如目标漂移、错误级联、信息共享不足和行为失调）而导致性能受限。
*   **研究动机**：借鉴人类心理学中的“社会评价”机制。人类在协作时会根据他人的表现形成关于“热情（Warmth）”和“能力（Competence）”的印象，从而决定信任谁、如何分配任务。本文旨在探索 LLM 智能体是否也能通过显式推断伙伴的稳定行为特质来优化协作。

### 2. 方法论：显式特质推理（ETI）
*   **核心思想**：为智能体提供一种脚手架机制，使其能够从交互历史中提取、维护并利用合作伙伴的结构化特质画像。
*   **关键技术细节**：
    *   **双维度框架**：将特质分为**热情**（目标一致性、协作性、可靠性、恶意性）和**能力**（执行能力、可靠性、适应性、效率）两个维度，共 8 个具体特质。
    *   **推理流程**：
        1.  **观察**：记录交互历史（动作、沟通、结果）。
        2.  **推断**：在每次交互后，提示模型根据历史为伙伴打分（1-7 分）并提供文字证据（JSON 格式）。
        3.  **注入**：将生成的特质画像附加到智能体的上下文提示词中。
        4.  **决策**：智能体利用这些画像调整后续的规划、任务分配和沟通策略。
*   **算法特点**：无需微调（Prompt-based），轻量级且具有领域通用性。

### 3. 实验设计
*   **实验场景**：
    1.  **受控环境（经济博弈）**：迭代囚徒困境（PD）和迭代猎鹿博弈（SH）。引入了“能力参数”来模拟执行失败，测试智能体在噪声环境下的推断准确性。
    2.  **复杂现实环境（MultiAgentBench）**：包括代码协作（Coding）、研究提案编写（Research）、商务谈判（Bargaining）和社交博弈（Werewolf/狼人杀）。
*   **对比基准（Benchmark）**：
    *   **CoT Baseline**：标准的思维链提示，包含任务描述和历史记录，但不进行显式特质推理。
    *   **任务特定脚手架**：如 MultiAgentBench 原有的角色扮演和推理指令。
*   **对比模型**：主要使用 Qwen3-8B 和 GPT-4o-mini。

### 4. 资源与算力
*   **硬件设施**：使用 AWS EC2 g5.12xlarge 实例，配备 **4 张 NVIDIA A10G GPU**（共 96GB 显存）。

*   **API 使用**：GPT-4o-mini 实验通过 OpenAI API 完成。

### 5. 实验数量与充分性
*   **实验规模**：
    *   经济博弈：每种配置运行 25 次，每场博弈持续 50 轮。
    *   MultiAgentBench：每个场景运行 **100 次随机种子实验**，确保统计显著性。
*   **充分性评估**：实验设计非常充分。除了主实验外，还进行了：
    *   **跨模型测试**：用 Qwen 生成特质供 GPT 使用，反之亦然。
    *   **消融/鲁棒性分析**：测试了特质预校准、跨任务迁移能力以及在伙伴行为突变时的适应性。
    *   **预测性分析**：通过逻辑回归证明了推断的特质确实能预测智能体的投票和决策行为。

### 6. 主要结论与发现
*   **性能显著提升**：在经济博弈中，ETI 将收益损失降低了 **45-77%**；在 MultiAgentBench 中，性能提升了 **3-29%**，协作质量提升了 **6-42%**。
*   **可靠的推断能力**：LLM 能够从纯行为历史中准确推断伙伴的潜在特质（如在狼人杀中识别反派）。
*   **模型差异**：Qwen3-8B 生成的特质画像比 GPT-4o-mini 更具区分度（方差更大），而 GPT 存在明显的“积极偏见（Positivity Bias）”，倾向于给高分，这反而限制了协作效果。
*   **特质的稳定性与迁移性**：推断出的特质在同一伙伴身上具有跨任务的参考价值。

### 7. 优点
*   **理论扎实**：基于成熟的社会心理学框架（温暖-能力模型），而非凭空设计。
*   **即插即用**：作为一种提示工程方法，不需要昂贵的模型微调。
*   **可解释性强**：通过显式的评分和证据，开发者可以清晰看到智能体是如何“看待”其伙伴的。
*   **鲁棒性**：在存在执行噪声和不确定性的环境中表现优于传统方法。

### 8. 不足与局限
*   **模型偏见风险**：ETI 会继承底层 LLM 的偏见（如 GPT 的讨好型人格或积极偏见），可能导致推断失准。
*   **适应滞后性**：由于 ETI 强调“稳定特质”，当合作伙伴突然改变行为模式（如从合作转为背叛）时，智能体可能需要较长时间才能更新画像。
*   **Token 开销**：虽然是轻量级的，但在长对话中生成和维护多个伙伴的画像会增加 Token 消耗（实验显示在某些场景下 Token 增加约 2 倍）。
*   **覆盖范围**：尚未在超大规模智能体集群（如数百个智能体）或异构架构（非 LLM 智能体混合）中进行测试。
