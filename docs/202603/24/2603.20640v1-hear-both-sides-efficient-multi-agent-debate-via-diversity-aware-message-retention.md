## 摘要
多智能体辩论（Multi-Agent Debate）已成为一种通过智能体间的迭代通信来提高大语言模型推理质量的有前景的框架。然而，在每一轮中广播所有智能体的消息会引入噪声和冗余，这可能会降低辩论质量并浪费计算资源。目前的方法通常依赖于不确定性估计，在广播前过滤低置信度的响应，但由于置信度分数校准不当以及对阈值选择的敏感性，这种方法并不可靠。为此，我们提出了多样性感知保留（Diversity-Aware Retention, DAR），这是一种轻量级的辩论框架，它在每一轮辩论中，在广播前选择与彼此以及与多数投票结果分歧最大的智能体响应子集。通过一种显式的基于索引的保留机制，DAR 在不进行修改的情况下保留原始消息，确保保留的分歧保持真实性。在多种推理和问答基准测试上的实验表明，我们的选择性消息传播一致地提高了辩论性能，特别是在智能体数量增加、噪声累积最严重的情况下。我们的结果强调，在多智能体推理系统中，智能体“听到”什么与智能体“说”什么同样重要。

## Abstract
Multi-Agent Debate has emerged as a promising framework for improving the reasoning quality of large language models through iterative inter-agent communication. However, broadcasting all agent messages at every round introduces noise and redundancy that can degrade debate quality and waste computational resources. Current approaches rely on uncertainty estimation to filter low-confidence responses before broadcasting, but this approach is unreliable due to miscalibrated confidence scores and sensitivity to threshold selection. To address this, we propose Diversity-Aware Retention (DAR), a lightweight debate framework that, at each debate round, selects the subset of agent responses that maximally disagree with each other and with the majority vote before broadcasting. Through an explicit index-based retention mechanism, DAR preserves the original messages without modification, ensuring that retained disagreements remain authentic. Experiments on diverse reasoning and question answering benchmarks demonstrate that our selective message propagation consistently improves debate performance, particularly as the number of agents scales, where noise accumulation is most severe. Our results highlight that what agents hear is as important as what agents say in multi-agent reasoning systems.

*   **研究背景**：多智能体辩论（Multi-Agent Debate, MAD）通过多个大语言模型（LLM）智能体之间的迭代交流来提升推理能力。
*   **核心问题**：传统的 MAD 框架在每一轮都会广播所有智能体的消息。随着智能体数量和辩论轮数的增加，这种“全量广播”会导致**信息冗余**（多个智能体思路雷同）和**噪声累积**（错误或低质量回答误导他人），不仅浪费计算资源（Token 消耗），还会降低最终的推理质量。
*   **研究动机**：现有的过滤方法多基于“置信度/不确定性”，但 LLM 的置信度往往校准不准。作者认为，辩论的价值在于“分歧”，保留那些与主流意见不同且彼此互补的观点比单纯保留高置信度观点更有效。

### 2. 论文提出的方法论：DAR 框架
核心思想是**多样性感知保留（Diversity-Aware Retention, DAR）**，即在每一轮辩论中，只选择并传播那些与彼此以及与多数意见分歧最大的响应子集。

*   **关键技术细节**：
    1.  **不确定性注入**：计算每个智能体响应的平均负对数似然（ANLL）作为不确定性分数，并将其附加到 Prompt 中，帮助智能体评估他人回答的可靠性。
    2.  **多数投票锚点**：在每一轮 Prompt 中预置上一轮的多数投票结果（Majority Vote），作为共识参考点，引导智能体进行验证或反驳。
    3.  **多样性过滤模块（Filter Agent）**：引入一个轻量级的过滤智能体（通常由相同的 LLM 担任），它接收所有响应，输出一组智能体索引（IDs），这些索引对应的响应应满足“彼此分歧最大”且“与多数意见分歧最大”。
    4.  **基于索引的保留**：过滤模块仅输出索引，系统直接提取原始消息进行广播。这种设计确保了消息的真实性，避免了 LLM 二次编辑带来的信息失真。

### 6. 主要结论与发现
*   **性能提升**：DAR 在所有基准测试中一致优于标准 MAD 和基于置信度的过滤方法。
*   **扩展性优势**：随着智能体数量 $N$ 的增加，标准 MAD 的性能往往会因噪声而饱和甚至下降，而 DAR 能够持续从更多智能体中获益。
*   **多样性的价值**：实验证明，保留具有分歧的推理路径比保留一致的路径更能显著提高辩论的准确率。

### 7. 优点
*   **高效轻量**：无需对模型进行微调，直接在推理阶段通过 Prompt 工程和简单的过滤逻辑实现。
*   **真实性保障**：通过“索引保留”机制，确保了智能体听到的原始分歧意见不被篡改。
*   **鲁棒性强**：不依赖于难以校准的绝对置信度阈值，而是利用相对的分歧程度进行筛选。

### 8. 不足与局限
*   **过滤成本**：虽然过滤模块是轻量级的，但它仍然需要一次额外的 LLM 调用来选择索引，在极大规模智能体场景下可能存在延迟。
*   **小模型限制**：在消融实验中发现，极小规模的模型（如 1.5B）在执行复杂的过滤逻辑时，准确性可能不如大模型，存在误删正确少数派意见的风险。
*   **任务局限**：主要集中在具有客观答案的推理和问答任务上，对于开放式创作或主观性极强的任务，分歧的定义和过滤效果尚待验证。
