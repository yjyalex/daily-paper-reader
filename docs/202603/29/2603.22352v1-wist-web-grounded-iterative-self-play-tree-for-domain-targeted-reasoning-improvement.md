## 摘要
具有可验证奖励的强化学习（RLVR）的最新进展为语言模型的自我提升提供了一条切实可行的路径，但现有方法面临着一个关键的权衡：内源性自博弈（endogenous self-play）可能会随着迭代而产生漂移，而基于语料库的方法则依赖于精心策划的数据环境。我们提出了 WIST，一个用于领域定向推理改进的基于 Web 的迭代自博弈树框架，它直接从开放网络中学习，无需任何预先安排的领域语料库。WIST 增量地扩展领域树以进行探索，并检索和清洗路径一致的 Web 语料库，以构建一个可控的训练环境。随后，它利用可验证的奖励进行“挑战者-求解者”（Challenger--Solver）自博弈，并将可学习性信号反馈以更新节点后验，并通过自适应课程指导后续探索。在四个骨干模型上，WIST 相比基础模型均有一致的提升，且通常优于纯内源性自我进化和基于语料库的自博弈基准，总体增益分别达到 +9.8（Qwen3-4B-Base）和 +9.7（OctoThinker-8B）。WIST 还具有领域可控性，使 Qwen3-8B-Base 在医学领域提升了 +14.79，使 Qwen3-4B-Base 在 PhyBench 上提升了 +5.28。消融实验进一步证实了 WIST 关键组件对于稳定开放网络学习的重要性。我们的代码可在 https://github.com/lfy-123/WIST 获取。

## Abstract
Recent progress in reinforcement learning with verifiable rewards (RLVR) offers a practical path to self-improvement of language models, but existing methods face a key trade-off: endogenous self-play can drift over iterations, while corpus-grounded approaches rely on curated data environments. We present \textbf{WIST}, a \textbf{W}eb-grounded \textbf{I}terative \textbf{S}elf-play \textbf{T}ree framework for domain-targeted reasoning improvement that learns directly from the open web without requiring any pre-arranged domain corpus. WIST incrementally expands a domain tree for exploration, and retrieves and cleans path-consistent web corpus to construct a controllable training environment. It then performs Challenger--Solver self-play with verifiable rewards, and feeds learnability signals back to update node posteriors and guide subsequent exploration through an adaptive curriculum. Across four backbones, WIST consistently improves over the base models and typically outperforms both purely endogenous self-evolution and corpus-grounded self-play baselines, with the Overall gains reaching \textbf{+9.8} (\textit{Qwen3-4B-Base}) and \textbf{+9.7} (\textit{OctoThinker-8B}). WIST is also domain-steerable, improving \textit{Qwen3-8B-Base} by \textbf{+14.79} in medicine and \textit{Qwen3-4B-Base} by \textbf{+5.28} on PhyBench. Ablations further confirm the importance of WIST's key components for stable open-web learning. Our Code is available at https://github.com/lfy-123/WIST.

### 论文总结：WIST —— 基于 Web 的迭代自博弈树框架

#### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）的自我提升目前主要依赖**具有可验证奖励的强化学习（RLVR）**。然而，现有方法存在两难困境：
*   **内生自博弈（如 R-Zero）：** 完全依赖模型生成任务，容易随迭代产生语义漂移和幻觉累积。
*   **语料驱动自博弈（如 SPICE）：** 依赖预先清洗的高质量领域语料库，难以扩展到缺乏现成数据的专业领域。
**WIST 的核心目标**是利用**开放互联网（Open Web）**作为训练环境，通过动态构建领域知识树和自动化检索清洗，实现无需人工标注语料的领域定向推理能力提升。

#### 2. 方法论：核心思想与技术细节
WIST 框架由三个相互耦合的组件组成：
*   **动态领域树构建（Dynamic Domain Tree）：**
    *   从用户指定的领域标签（如“数学”）开始，模型递归地将大主题分解为子主题，直至叶子节点的“知识点”。
    *   引入 `unk`（未知）占位符节点，当采样到该节点时触发模型生成新子主题，实现树的增量扩展。
    *   利用 Wikipedia 标题匹配进行非叶子节点的合法性验证，防止幻觉概念污染。
*   **Web 检索与语料构建：**
    *   以叶子节点为关键词检索 Web 文档，通过 URL 黑白名单、语义相似度过滤和正文清洗（去除广告、导航栏等），为每个知识点构建轻量级语料池。
*   **挑战者-求解者（Challenger-Solver）自博弈：**
    *   **Challenger（挑战者）：** 根据检索到的文档生成可验证的 QA 对。
    *   **Solver（求解者）：** 在不看文档的情况下回答问题。
    *   **奖励设计：** Solver 获得基于正确性的二元奖励；Challenger 获得基于“难度建模”的奖励（倾向于生成 Solver 表现出中等方差、即处于能力边界的问题）。
*   **后验引导的课程学习：**
    *   利用 Beta 分布维护每个节点的“可学习性”后验。
    *   通过汤普森采样（Thompson Sampling）选择路径，并使用滑动窗口更新统计信息，使探索过程动态聚焦于模型尚未掌握但可学习的区域。

#### 7. 优点（亮点）
*   **摆脱语料依赖：** 创新性地将 Open Web 视为动态环境，解决了专业领域高质量语料匮乏的问题。
*   **动态课程学习：** 通过领域树和后验更新，实现了自动化的“由易到难”或“针对弱点”的训练路径规划。
*   **闭环进化：** 实现了从“探索（树扩展）-> 获取（Web 检索）-> 学习（自博弈）-> 反馈（后验更新）”的完整闭环。

#### 8. 不足与局限
*   **验证器依赖：** 核心仍依赖“可验证奖励”（RLVR），对于法律、创意写作等难以自动判定对错的领域，该框架的应用受到限制。
*   **Web 噪声风险：** 尽管有清洗流程，但互联网数据的质量波动仍可能引入错误知识，尤其是对判别能力较弱的小模型。
*   **算力开销：** 动态检索和频繁的自博弈采样（Challenger 和 Solver 各采样 8 次）带来了较高的推理计算成本。
