## 摘要
强化学习通过奖励信号直接优化模型行为，从而改进大语言模型。尽管准确的状态价值估计对于经典强化学习中的稳定训练至关重要，但在LLM后训练中，这仍然是一个未被充分探索的挑战。在这项工作中，我们引入了状态价值估计基准（SVEB），用于评估现有强化学习框架中的状态估计，并展示了在PPO等标准方法中，评论者会退化为粗糙的组平均基线。为了解决这个问题，我们提出了两种技术：Numca，它利用数值跨度作为可分级里程碑进行状态价值估计；以及Hista，一个使用LLM隐藏状态作为表示来加权平均不相交轨迹及其回报的框架。大量实验表明，这两种方法都能产生更准确的状态价值估计，并在不同的强化学习算法和模型规模下提升训练性能，且不会带来显著的计算开销。

## Abstract
Reinforcement learning (RL) refines large language models (LLMs) by directly optimizing model behavior through reward signals. While accurate state value estimation is critical for stable training in classical RL, it remains an underexplored challenge in LLM post-training. In this work, we introduce the State Value Estimation Benchmark (SVEB) to assess state estimation within existing RL frameworks and show that critics in standard approaches like PPO collapse to a coarse group-average baseline. To address this, we propose two techniques: Numca, which leverages numerical spans as gradable milestones for state value estimation, and Hista, a framework that uses LLM's hidden states as representation to weighted average disjoint rollouts and their return. Extensive experiments demonstrate that both methods yield more accurate state value estimates and enhance training performance across different RL algorithms and model sizes without incurring significant computational overhead.

- **问题**：在大语言模型（LLM）的强化学习（RL）后训练中，准确的状态价值估计对稳定训练至关重要，但当前主流方法（如GRPO）将整个响应视为单个动作，对所有 token 使用相同的组平均奖励，导致信用分配粗糙。即便使用 PPO 的 critic 网络，其价值估计也会退化为组平均基线，无法提供细粒度的指导。
- **动机**：回答“准确的状态价值估计如何改善 LLM RL 训练，以及如何有效估计它”这一核心问题。
- **背景**：经典 RL 中 critic 网络提供细粒度价值，但在 LLM 横生场景下，PPO 的 critic 因初始化问题和序列长度复杂性而失效。现有改进（如过程奖励模型、蒙特卡洛树搜索）成本过高或依赖领域知识。

### 2. 论文提出的方法论

- **Numca（Numerical Milestone Credit Assignment）**  
  - **核心思想**：将数学推理视为稀疏奖励的目标达成问题，利用数字（整数、小数、分数等）作为自然可验证的子目标（milestone）。  
  - **技术细节**：  
    - 定义抽象状态 \(s^M_t\) 为当前已到达的所有数字里程碑集合。  
    - 定义宏动作（macro action）为两个连续抽象状态间的 token 序列。  
    - 状态价值估计：对同一抽象状态 \(s^M\) 的所有 rollout 的最终奖励取平均。  
    - 该价值均匀分配给对应宏动作内的所有 token。  
    - 算法使用字典实现，几乎无额外内存和计算开销。  

- **Hista（Hidden State based State Value Estimation）**  
  - **核心思想**：利用 LLM 的隐藏状态作为通用状态表示，通过加权平均不交叉轨迹的回报来估计当前状态价值，无需额外训练或 rollout。  
  - **技术细节**：  
    - **隐藏状态提取**：获取最后解码器层的隐藏状态序列 \(X_\tau\)。  
    - **压缩与池化**：使用指数移动平均（EMA，平滑因子 \(\alpha\)）和压缩间隔 \(\varphi\) 将长序列压缩为较短嵌入序列 \(E_\tau\)。  
    - **状态采样**：以间隔 \(\delta\) 均匀采样嵌入位置，定义有限状态空间 \(S^H_\tau\)。  
    - **距离计算**：采用 MinDistance（MD）操作计算两个状态表示之间的最小 L2 距离（见定义 5.1）。  
    - **价值估计**：对当前状态 \(s^H_t\)，找到与其 MD 最小的 k 个邻居，以 MD 的倒数作为权重，加权平均这些邻居的最终奖励：  
      \[
      V(s_t) = \frac{\sum_{i=1}^k \omega_i \cdot r_i}{\sum_{i=1}^k \omega_i}, \quad \omega_i = \frac{1}{\mathrm{MD}(s_t, s_i)}.
      \]  
    - **理论保证**：定理 5.2 证明隐藏状态 MD 与两个状态获得相同奖励的概率成正比；定理 5.5 证明概率加权估计器比简单组平均估计器有更小的绝对偏差（满足一定假设下）。

### 6. 论文的主要结论与发现

- **SVEB 揭示**：标准 PPO 的 critic 在 LLM 场景下价值估计退化到组平均基线，无法提供有效指导。  
- **Numca**：在数学推理任务上显著优于 GRPO 和 PPO-N（SVEB-Number 上 MAE 0.132 vs GRPO 0.175），且 RL 训练更稳定，但依赖数值先验，无法推广到科学等其他领域。  
- **Hista**：在所有 SVEB 领域上一致优于 GRPO 和 PPO-N，MAE 降低约 0.02–0.04，逼近 MCS@2 精度。与 DAPO/CSIPO 结合后，在数学、科学、编程等多个下游基准上获得稳定提升（平均 +0.5%–1.7%）。理论上证明了概率加权估计器比组平均估计器有更小的偏差。  
- **效率**：Hista 计算开销与 GRPO 相当，额外延迟 <1s/步。  

### 7. 优点

- **问题定位精准**：通过构建 SVEB 基准，首次清晰地量化了 PPO critic 在 LLM RL 中的退化现象。

### 8. 不足与局限
