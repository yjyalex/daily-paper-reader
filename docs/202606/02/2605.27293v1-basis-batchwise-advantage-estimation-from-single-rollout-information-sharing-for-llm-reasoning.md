## 摘要
基于可验证奖励的强化学习已成为提升大语言模型推理能力的标准方法。现有算法在价值估计和策略学习上面临计算效率与样本效率之间的权衡。我们提出BASIS，一种无需评论家的后训练算法，旨在解决这一权衡。在每个在线训练步骤中，BASIS每个提示仅采样一次，但利用整个批次中跨提示的丰富信息来改进价值函数估计。实验表明，与代表性单次采样基线REINFORCE++相比，BASIS将价值函数估计的均方误差降低了69%，并且使用一次采样达到的均方误差低于使用8次采样的组均值估计器。这种价值估计的提升转化为更好的策略优化：使用显著更少的训练时间，BASIS达到了接近多次采样GRPO型基线的性能，并且通常优于单次采样REINFORCE型基线。

## Abstract
Reinforcement learning with verifiable rewards has become a standard recipe for improving the reasoning abilities of large language models. Existing algorithms face a tradeoff between computational efficiency and sample efficiency in value estimation and policy learning. We introduce BASIS, a critic-free post-training algorithm designed to address this tradeoff. At each online training step, BASIS samples only one rollout per prompt, but leverages rich information across prompts in the entire batch to improve value function estimation. Our experiments demonstrate that BASIS reduces MSE in value function estimation by 69% compared to REINFORCE++, a representative single-rollout baseline, and achieves lower MSE with one rollout than group mean estimators with 8 rollouts. This improvement in value estimation translates to better policy optimization: using substantially less training time, BASIS achieves performance close to multi-rollout GRPO-type baselines and often outperforms single-rollout REINFORCE-type baselines.

- **研究背景**：基于可验证奖励的强化学习（RLVR）已成为提升大语言模型推理能力的标准后训练方法。现有 RLVR 算法在价值/优势估计中面临计算效率与样本效率的根本权衡。
- **核心问题**：
  - PPO 类算法通过训练评论家网络（value network）获得精准估计，但计算成本高。
  - GRPO 类算法通过多轮采样（per‑prompt multiple rollouts）用组均值近似价值，无需评论家，但多次采样增加了显著的计算开销。
  - 单次采样算法（如 REINFORCE++）计算成本低，但价值估计粗糙，导致策略更新噪声大、易崩溃。
- **本文目标**：提出一种**无需评论家、单 rollouts 采样**的算法 BASIS，通过跨 prompt 共享批次内奖励信息，在保持计算高效的同时提升价值估计精度，从而接近多 rollouts 方法的样本效率。

## 2. 方法论

### 核心思想
- 每个训练步骤每 prompt 仅采样一个 rollout，但利用整个 batch 中其他 prompt 的奖励信息来构造每个 prompt 的价值基线。
- 基线由加权平均构成，权重通过最小化 MSE 确定，且采用 leave‑one‑out 原则（排除自身 reward）。
- 方法分为三步：**离线值估计 → 批处理细化 → 在线校准**。

### 关键技术细节

1. **离线值估计（Offline Value Estimation）**  
   - 基于 KL 正则化目标（参考策略 π_ref）得到的最优策略 π*_β 的值函数存在闭式解：
     \[
     V^*_\beta(x) = \frac{\mathbb{E}_{y\sim\pi_{\text{ref}}(\cdot|x)}[r(x,y)\exp(r(x,y)/\beta)]}{\mathbb{E}_{y\sim\pi_{\text{ref}}(\cdot|x)}[\exp(r(x,y)/\beta)]}
     \]
   - 对每个 prompt 从参考策略采样多组 rollout，用经验均值估计上述期望，得到初始离线估计 \(\hat{V}_\beta(x_i)\)（二值奖励时方差估计为 \(\hat{\sigma}_i^2 = \hat{V}_{i,t}(1-\hat{V}_{i,t})\)）。

2. **批处理细化（Batchwise Refinement）**  
   - 对每个 prompt \(i\)，利用 batch 内其他 prompt 的奖励进行加权平均构造精细估计：
     \[
     \tilde{V}_{i,t} = \sum_{j\neq i} w_{ij} r_j
     \]
   - 权重由**最优线性无偏估计（BLUE）** 公式给出：
     \[
     w_{ij} \propto \frac{V_{i,t} V_{j,t}}{\sigma_j^2}
     \]
     实际中替换为离线估计值。该估计在无偏条件下最小化条件 MSE。
   - 为避免数值不稳定，定义活动集 \(A_t = \{i: \epsilon < \hat{V}_{i,t} < 1-\epsilon\}\)，仅对活动内 prompt 应用该估计，其余用零基线回退。

3. **在线校准（Online Calibration）**  
   - 使用一族不同 β 的离线估计 \(\{V^*_\beta(x)\}_{\beta\in\mathcal{B}}\) 近似当前策略的值函数，β 在每步自适应选择：
     \[
     \beta_t = \arg\min_{\beta\in\mathcal{B}} \frac{1}{|A_{t,\beta}|}\sum_{i\in A_{t,\beta}} (r_i - \tilde{V}_{i,t,\beta})^2
     \]
   - β 随训练从大变小（对应 KL 权重减弱），算法 1 总结了完整流程。

- **BASIS 作为插件**：其优势估计可直接嵌入 GRPO、GPG、GSPO 等算法，仅替换基线部分。
