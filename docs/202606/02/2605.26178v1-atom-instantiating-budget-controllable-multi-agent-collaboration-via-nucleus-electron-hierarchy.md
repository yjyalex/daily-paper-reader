## 摘要
基于大型语言模型的多智能体系统依赖于优化的协作拓扑以平衡性能与通信成本。然而，当前方法难以解决固有的稳定性-可扩展性权衡问题，并且常常将计算预算与查询难度错配。我们提出ATOM，一种通过新颖的任务驱动强化学习范式生成预算可控协作图的自适应框架。受原子结构启发，ATOM采用核-电子层级：它维护一个稳定的、离线学习的协作主干（原子核），同时在推理过程中动态激活查询条件驱动的智能体（电子）。关键地，一种复杂度感知的预算策略通过估计查询难度来严格调控电子实例化，从而将资源消耗与任务需求对齐。在六个多样化基准上的大量实验表明，ATOM达到了最先进的性能，同时与强基线相比，令牌效率提高了高达30%。

## Abstract
Large Language Model (LLM)-based multi-agent systems rely on optimized collaboration topologies to balance performance and communication costs. However, current methods struggle with the inherent stability-extensibility trade-off and often misalign computational budgets with query difficulty. We propose \textsc{ATOM}, an adaptive framework that generates budget-controllable collaboration graphs via a novel task-driven reinforcement learning paradigm. Inspired by atomic structures, \textsc{ATOM} employs a nucleus-electron hierarchy: it maintains a stable, offline-learned collaboration backbone (the nucleus) while dynamically activating query-conditioned agents (electrons) during inference. Crucially, a complexity-aware budgeting strategy aligns resource consumption with task demands by estimating query difficulty to strictly regulate electron instantiation. Extensive experiments across six diverse benchmarks demonstrate that \textsc{ATOM} achieves state-of-the-art performance while improving token efficiency by up to $30\%$ compared to strong baselines.

## 2. 论文提出的方法论

### 核心思想：核-电子层级（Nucleus-Electron Hierarchy）

受原子结构启发，将MAS组织为“原子”：  
- **核（Nucleus）**：一个离线学习、稳定的智能体骨干子集，充当可靠的推理主干；  
- **电子（Electrons）**：在线动态激活的查询条件智能体，用于按需扩展通信。  
所有智能体间的通信边（包括核内部）仍为动态、查询条件驱动，以保持灵活性。

### 关键技术细节

1. **两阶段拓扑学习**  
   - **离线阶段**：在一个全连接超网络上，利用随机边参数化（logits）和REINFORCE算法优化结构，同时施加稀疏性正则化。收敛后，根据每个智能体的期望度中心性（EDC）选出前K个作为稳定核，其余为电子池。  
   - **在线阶段**：给定查询，先估计复杂度，确定电子预算上限 \(K(q)\)；然后通过条件伯努利采样激活不超过预算的电子子集；最后基于激活的节点，动态生成空间边（有向无环图DAG约束）和时间边（跨轮记忆检索）。

2. **复杂度感知预算策略**  
   - 构建复合特征向量 \(x_q = [\text{WPCA}(f_{\text{enc}}(q)) \parallel \Phi(q)]\)，其中 \(\Phi(q)\) 量化符号/逻辑密度（如变量数、逻辑运算符数）。通过领域门控机制消除跨域混淆，用MLP回归得到归一化复杂度 \(\bar{C}(q) \in [0,1]\)，进而设定电子预算 \(K(q) = \lfloor K_{\max} \cdot \bar{C}(q) \rfloor\)。

3. **任务驱动强化学习训练**  
   - 优化目标：最大化任务效用，同时受预算约束和结构正则化。奖励函数 \(R(G) = r_{\text{task}} - \lambda_{\text{cost}} (|E_{\text{active}}|/|E_{\max}|)\)。  
   - 结构正则化 \(L_{\text{struct}}\)：双边际排名损失，迫使路由偏向核中心通信（即核-核、核-电子边的概率高于电子-电子边）。  
   - 使用标准化优势、长度归一化序列代理（处理DAG约束）来稳定梯度。

### 算法流程简述
1. 离线学习：优化全连接超网边logits → 计算节点重要性 → 划分核/电子。
2. 在线推理：查询输入 → 复杂度估计 → 计算电子预算 → 条件伯努利采样激活电子 → 条件DAG生成空间边 + 独立采样时间边 → 执行多轮对话 → 得到答案并更新策略（RL训练）。

- **理论驱动的预算控制**：通过复杂度估计显式对齐计算资源与任务难度，避免了平均复杂度陷阱，实现真正按需分配。
