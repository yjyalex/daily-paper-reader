## 摘要
当语言模型回答表格问题时，用户无法验证哪些单元格支撑了哪些推理步骤。我们提出了 RSAT，这是一种训练小语言模型（SLM，1-8B）的方法，使其能够生成带有基于表格证据的单元格级引用的逐步推理。第一阶段（SFT）通过经过验证的推理轨迹教授结构化 JSON 输出格式。第二阶段（GRPO）优化以基于自然语言推理（NLI）的忠实度为核心的复合奖励，同时兼顾引用的有效性和简洁性。在来自两个系列的六个模型——Qwen 2.5 (1.5B/3B/7B) 和 Llama 3 (1B/3B/8B) 上，RSAT 将忠实度比仅使用 SFT 提高了 3.7 倍（从 0.224 提升至 0.826），且引用有效性接近完美（0.992）。事后归因（Post-hoc attribution）的格式成功率跌至 13% 以下，这证实了归因必须整合到推理过程中，而不是事后补救。消融实验表明，忠实度奖励至关重要：移除它会导致忠实度从 0.97 降至 0.03。

## Abstract
When a language model answers a table question, users have no way to verify which cells informed which reasoning steps. We introduce RSAT, a method that trains small language models (SLMs, 1-8B) to produce step-by-step reasoning with cell-level citations grounded in table evidence. Phase 1 (SFT) teaches a structured JSON output format from verified reasoning traces. Phase 2 (GRPO) optimizes a composite reward centered on NLI-based faithfulness, alongside citation validity and parsimony. Across six models from two families-Qwen 2.5 (1.5B/3B/7B) and Llama 3 (1B/3B/8B)-RSAT improves faithfulness 3.7$\times$ over SFT alone (0.224$\rightarrow$0.826), with near-perfect citation validity (0.992). Post-hoc attribution collapses below 13% format success, confirming that attribution must be integrated into reasoning, not retrofitted. Ablations show the faithfulness reward is essential: removing it drops faithfulness from 0.97 to 0.03.

以下是对该论文的深度结构化总结：

### 1. 核心问题与研究动机
*   **核心问题**：当语言模型回答基于表格的问题时，用户无法验证推理步骤究竟引用了哪些具体的单元格。这种“黑盒”属性限制了模型在金融、医疗等高风险领域的应用。
*   **研究背景**：
    *   现有的表格问答方法（如 TAPAS, CoT）虽能提高准确率，但无法提供显式的、单元格级别的证据支撑。
    *   **事后归因（Post-hoc attribution）**（即先生成答案再补写引用）在参数量小于 8B 的小模型上表现极差，经常导致格式崩溃或引用错误。
*   **研究目标**：训练 1B-8B 规模的小模型，使其能够生成带有结构化 JSON 格式、且每一步推理都精准对应表格单元格坐标的“忠实”推理轨迹。

### 2. 方法论：RSAT 训练框架
RSAT 采用两阶段训练流程，核心思想是将“格式学习”与“质量优化”分离：

*   **第一阶段：监督微调 (SFT)**
    *   **目标**：教授模型结构化的 JSON 输出格式。
    *   **数据**：使用 Claude 4.5 生成 1,000 条经过验证的推理轨迹，包含 `reasoning_steps`（推理步骤）、`cited_cells`（引用单元格坐标 `[row, col]`）和 `answer`（最终答案）。
    *   **效果**：使模型获得近乎 100% 的格式合规性，但此时引用的“忠实度”较低（仅约 22%）。

*   **第二阶段：群体相对策略优化 (GRPO)**
    *   **核心技术**：使用 GRPO 算法（DeepSeek-R1 同款），无需 Critic 模型，节省显存，适合单 GPU 训练。
    *   **复合奖励函数 ($R$)**：
        1.  **答案奖励 ($R_{ans}$)**：预测答案与标准答案的 F1 分数。
        2.  **引用有效性 ($R_{cite}$)**：检查坐标是否在表格范围内。
        3.  **忠实度奖励 ($R_{faith}$)**：**核心创新**。使用 DeBERTa-v3 NLI 模型计算引用单元格内容与推理文本之间的蕴含概率。
        4.  **简洁性奖励 ($R_{pars}$)**：惩罚过度引用（如一次引用整个表格），鼓励精准引用。
        5.  **格式惩罚 ($R_{fmt}$)**：对非 JSON 输出进行硬性扣分。

### 3. 实验设计
*   **数据集**：涵盖了三个主流表格推理基准：**WTQ**（事实问答）、**FeTaQA**（自由简答）和 **TabFact**（事实验证）。总语料库约 7.5 万条。
*   **实验模型**：
    *   **Qwen 2.5 系列**：1.5B, 3B, 7B。
    *   **Llama 3 系列**：1B, 3B, 8B。
*   **对比方法 (Baselines)**：
    1.  **Zero-shot**：原始指令微调模型。
    2.  **SFT-only**：仅进行第一阶段微调。
    3.  **Post-hoc**：两步走方案（先推理，再由模型事后标注引用）。
    4.  **RSAT**：本文提出的完整两阶段方案。

*   **细节**：SFT 训练 3 个 Epoch，GRPO 训练 1 个 Epoch（约 250 个优化步数）。

### 5. 实验数量与充分性
*   **实验规模**：在两个架构家族、三个参数梯度（1B 到 8B）上进行了全面测试，实验覆盖面广。
*   **消融实验**：非常充分。作者分别移除了忠实度奖励、简洁性奖励和引用奖励，验证了各组件的必要性。
*   **客观性**：使用了相同的系统提示词和评估指标，并对比了事后归因这一强竞争思路，证明了“集成归因”的优越性。

### 6. 主要结论与发现
*   **忠实度飞跃**：RSAT 将模型的推理忠实度平均提升了 **3.7 倍**（从 0.224 提升至 0.826），且引用有效性保持在 0.99 以上。
*   **事后归因的崩溃**：在小模型上，事后归因的格式成功率不足 13%，证明了归因必须与推理同步生成。
*   **SFT 与 RL 的分工**：SFT 负责“学样子”（格式），GRPO 负责“学里子”（质量）。两者缺一不可。
*   **模型规模效应**：Qwen 3B 在忠实度与参数效率平衡上表现最佳；Llama 系列则需要达到 8B 才能接近 Qwen 的表现。
*   **关键信号**：消融实验显示，**忠实度奖励（NLI）是唯一不可或缺的信号**，移除它会导致忠实度从 0.97 暴跌至 0.03。

### 7. 论文优点
*   **高效性**：证明了通过轻量级 RL（GRPO）和少量 SFT 数据，即可在单卡上显著提升小模型的复杂推理能力。
*   **可解释性**：输出的 JSON 格式允许用户或自动化脚本对每一行推理进行“审计”，极大地增强了系统的可信度。
*   **方法论清晰**：成功将非微分的 NLI 分数转化为强化学习的奖励信号，解决了归因难以量化训练的问题。

### 8. 不足与局限
*   **评估闭环风险**：训练时的奖励模型（DeBERTa NLI）与评估时的指标使用同一个模型，可能存在“刷分”嫌疑，即模型学会了迎合该特定 NLI 模型的偏好。
*   **缺乏人类评价**：虽然 NLI 是很好的代理指标，但论文未提供人类对推理步骤准确性的主观评估。
*   **领域局限性**：实验主要集中在维基百科表格，尚未在金融报表、医疗记录等更复杂、更专业的垂直领域表格上验证泛化性。
*   **精确匹配率低**：由于模型倾向于对答案进行改写，Exact Match (EM) 指标极低，这在一定程度上反映了模型在严格遵循简短答案格式方面仍有提升空间。
