## 摘要
强化学习（RL）通过奖励信号直接优化模型行为来改进大语言模型（LLMs）。尽管准确的状态价值估计对于经典RL中的稳定训练至关重要，但在LLM后训练中，这一挑战仍未得到充分探索。本文引入状态价值估计基准（SVEB）来评估现有RL框架中的状态估计，并发现标准方法（如PPO）中的critic会退化为粗糙的组平均基线。为此，我们提出两种技术：Numca，它利用数值跨度作为可分级里程碑来进行状态价值估计；以及Hista，该框架利用LLM的隐藏状态作为表示，对不相交的轨迹及其回报进行加权平均。大量实验表明，这两种方法能够获得更准确的状态价值估计，并提升不同RL算法和模型规模下的训练性能，且不会带来显著的计算开销。

## Abstract
Reinforcement learning (RL) refines large language models (LLMs) by directly optimizing model behavior through reward signals. While accurate state value estimation is critical for stable training in classical RL, it remains an underexplored challenge in LLM post-training. In this work, we introduce the State Value Estimation Benchmark (SVEB) to assess state estimation within existing RL frameworks and show that critics in standard approaches like PPO collapse to a coarse group-average baseline. To address this, we propose two techniques: Numca, which leverages numerical spans as gradable milestones for state value estimation, and Hista, a framework that uses LLM's hidden states as representation to weighted average disjoint rollouts and their return. Extensive experiments demonstrate that both methods yield more accurate state value estimates and enhance training performance across different RL algorithms and model sizes without incurring significant computational overhead.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：强化学习（RL）已被广泛应用于大语言模型（LLM）的后训练，以优化模型行为。在经典 RL 中，准确的状态价值估计（state value estimation）对训练稳定性至关重要，然而在 LLM 后训练中，该问题尚未得到充分研究。
- **核心问题**：当前主流 LLM RL 方法（如 GRPO）将整个响应视为单个动作，对所有 token 使用相同的状态价值（即组平均奖励），导致信用分配粗糙。即便采用标准 PPO 这类 actor-critic 方法，其 critic 网络在 LLM 场景中也会退化为一个粗分组平均值基准，无法提供比组平均更多信息的引导。
- **研究意义**：揭示该退化现象，并探索如何更有效地估计状态价值，从而提升 LLM RL 训练的效率和性能。

## 2. 论文提出的方法论

### 2.1 核心思想
论文提出两种状态价值估计方法：
- **Numca**（Numerical Milestone Credit Assignment）：针对数学推理域，利用数值（如整数、小数、分数）作为自然可验证的里程碑，将抽象状态定义为里程碑集合，并通过汇总包含该里程碑轨迹的最终奖励来估计状态价值。
- **Hista**（Hidden State based State Value Estimation）：一种通用框架，利用 LLM 最后一层的隐藏状态作为状态表示，通过计算两个状态隐藏序列的 MinDistance（最小距离）来近似它们返回相同最终奖励的概率，然后对 k 个最近邻轨迹的回报进行概率加权平均，得到当前状态的价值估计。

### 2.2 关键技术细节

#### Numca
- 定义里程碑模式集合 P（如数字模式）。
- 对每个轨迹，提取里程碑序列，形成抽象状态 s^M（里程碑列表）。
- 维护字典 T，记录每个抽象状态的计数和奖励和。
- 估计值 V(s^M) = 奖励和 / 计数，并均匀分配给该抽象状态对应宏动作中的所有 token。
- 无需额外训练，高效（仅字典操作）。

#### Hista
- **隐藏状态提取**：从 LLM 最后一层提取 token 级隐藏状态序列。
- **压缩与采样**：使用指数移动平均（EMA）和压缩间隔 φ 压缩序列长度，再以间隔 δ 均匀采样位置，得到有限状态空间。
- **距离计算**：采用 MinDistance 操作，定义为两个隐藏状态序列中每个 token 到对方序列最近 token 距离之和（基于 ℓ2 范数）。
- **价值估计**：对目标状态，找到同组 prompt 下 k 个最近邻状态，以其最终奖励的概率加权平均作为估计值：V(s_t) = Σ (ω_i · r_i) / Σ ω_i，其中 ω_i = 1 / MD(s_t, s_i)。
- **理论保证**：定理 5.2 证明隐藏状态 MinDistance 与状态间最终奖励相同的概率成反比；定理 5.5 证明概率加权估计的绝对偏差小于组平均估计器。
- **计算效率**：通过 GPU 批量计算，空间复杂度 O(⌊T/δ⌋²)，时间开销与 GRPO 相当（＜1 秒/步）。

### 2.3 算法流程
- **SVEB 基准**：构建包含状态及其蒙特卡洛参考价值的数据集，以 MAE 作为评价指标。
- **训练流程**：将 Hista 或 Numca 嵌入现有 RL 算法（如 DAPO、CSIPO）中，在组内用估计价值替换组平均基线计算优势函数（A_t = r_t - V(s_t)），然后进行标准的策略梯度更新。

## 6. 论文的主要结论与发现

- **SVEB 揭示退化**：标准 PPO 的 critic 价值估计与组平均 GRPO 差异极小，分布集中在零附近，说明 critic 丧失了提供细粒度信号的能力。
- **Numca 有效但领域受限**：在数学推理领域，Numca 的 MAE 显著低于 GRPO 和 PPO-N，并能提升 RL 训练稳定性与最终性能；但在非数学领域（科学、通用等）表现退化为 GRPO 水平。

## 7. 优点

- **理论贡献**：建立了隐藏状态 MinDistance 与最终奖励相关性之间的理论联系，并证明了加权估计优于组平均。
- **方法新颖性**：将经典 RL 中的价值函数估计推广到 LLM 场景，利用 LLM 自身语义表示进行信用分配，无需额外训练或标注。
- **通用性**：Hista 适用于数学、科学、通用、编程等多个领域，且在不同模型架构（Qwen2.5/3）和不同初始化（Base/Instruct）上均有效。

## 8. 不足与局限

- **Numca 的领域依赖性**：需要预定义的数值模式，仅适用于数学推理，无法直接迁移到其他领域。
