## 摘要
当语言模型回答表格问题时，用户无法验证哪些单元格支撑了哪些推理步骤。我们提出了 RSAT，这是一种训练小语言模型（SLM，1-8B）的方法，使其能够生成基于表格证据的、带有单元格级引用的逐步推理。第一阶段（SFT）通过经过验证的推理轨迹教授结构化的 JSON 输出格式。第二阶段（GRPO）优化了一个以基于自然语言推理（NLI）的忠实度为核心的复合奖励，同时兼顾引用的有效性和简洁性。在来自两个系列的六个模型——Qwen 2.5 (1.5B/3B/7B) 和 Llama 3 (1B/3B/8B) 上，RSAT 将忠实度比单纯的 SFT 提高了 3.7 倍（从 0.224 提升至 0.826），且引用有效性接近完美（0.992）。事后归因（Post-hoc attribution）的格式成功率跌至 13% 以下，这证实了归因必须集成到推理过程中，而不是事后补救。消融实验表明，忠实度奖励至关重要：移除它会导致忠实度从 0.97 下降到 0.03。

## Abstract
When a language model answers a table question, users have no way to verify which cells informed which reasoning steps. We introduce RSAT, a method that trains small language models (SLMs, 1-8B) to produce step-by-step reasoning with cell-level citations grounded in table evidence. Phase 1 (SFT) teaches a structured JSON output format from verified reasoning traces. Phase 2 (GRPO) optimizes a composite reward centered on NLI-based faithfulness, alongside citation validity and parsimony. Across six models from two families-Qwen 2.5 (1.5B/3B/7B) and Llama 3 (1B/3B/8B)-RSAT improves faithfulness 3.7$\times$ over SFT alone (0.224$\rightarrow$0.826), with near-perfect citation validity (0.992). Post-hoc attribution collapses below 13% format success, confirming that attribution must be integrated into reasoning, not retrofitted. Ablations show the faithfulness reward is essential: removing it drops faithfulness from 0.97 to 0.03.

以下是对该论文的结构化总结：

### 1. 核心问题与研究背景
*   **核心问题**：当语言模型回答基于表格的问题时，用户难以验证其推理步骤是否真的基于表格中的特定数据。现有的方法虽然准确率高，但缺乏单元格级别的显式归因。
*   **研究背景**：
    *   **可解释性缺失**：在金融、医疗等高风险领域，无法验证的推理结果限制了模型的使用。
    *   **事后归因（Post-hoc）的局限**：先生成答案再补写引用的方式在参数量小于 8B 的小模型上表现极差，容易出现格式崩溃或引用虚假。
    *   **研究动机**：探索如何通过训练让小模型在生成推理步骤的同时，产出精确、忠实的单元格引用。

### 2. 方法论（RSAT）
RSAT 采用两阶段训练流程，将归因能力集成到推理过程中：
*   **核心思想**：将表格推理任务转化为生成带有 `[row, col]` 坐标引用的结构化 JSON 格式，并通过强化学习优化引用的质量。
*   **第一阶段：监督微调（SFT）**：
    *   使用 Claude 4.5 生成 1,000 条高质量推理轨迹作为训练集。
    *   目标是教会模型输出特定的 JSON 结构（包含 `reasoning_steps`、`cited_cells` 和 `answer`）。
    *   **结果**：模型学会了格式，但引用的单元格往往与推理内容无关（忠实度低）。
*   **第二阶段：组相对策略优化（GRPO）**：
    *   使用 GRPO 算法（无需 Critic 模型，节省显存）进行强化学习。
    *   **复合奖励函数**：
        1.  **答案奖励 ($R_{ans}$)**：预测答案与标准答案的 F1 分数。
        2.  **引用有效性 ($R_{cite}$)**：引用坐标是否在表格范围内。
        3.  **忠实度 ($R_{faith}$)**：**核心创新点**。使用 DeBERTa-v3 NLI 模型计算引用单元格内容与推理步骤文本之间的蕴含概率。
        4.  **简洁性 ($R_{pars}$)**：惩罚过度引用（如引用整个表格），鼓励精准引用。
        5.  **格式惩罚 ($R_{fmt}$)**：若 JSON 解析失败则给予重罚。

*   **数据集**：涵盖了三个主流表格推理基准：

    *   **TabFact**：表格事实验证。

    *   **SFT-only**：仅进行第一阶段微调。

### 5. 实验数量与充分性
*   **实验规模**：针对 2 个模型家族、3 种参数规模、3 个数据集进行了全面测试。
*   **消融实验**：专门针对奖励函数的各个组件（忠实度、简洁性、引用有效性）进行了剔除实验，验证了各部分的必要性。
*   **客观性**：实验对比了多种基准方法，并展示了模型在不同规模下的扩展性曲线（Scaling Curves），数据支撑详实，结论较为客观。

### 6. 主要结论与发现
*   **忠实度飞跃**：RSAT 将平均忠实度从 SFT 的 0.224 提升至 **0.826**（提升 3.7 倍），且引用有效性接近 100%。
*   **事后归因崩溃**：在小模型上，事后归因的格式成功率不足 13%，证明归因必须与推理**同步生成**。
*   **SFT 与 RL 的分工**：SFT 负责教会模型“格式”，而 GRPO 负责教会模型“质量（忠实度）”。
*   **模型差异**：在同等参数规模下，Qwen 系列在表格归因任务上的表现优于 Llama 系列。
*   **忠实度奖励是关键**：消融实验显示，移除 NLI 忠实度奖励后，模型会通过随机引用单元格来“刷分”，导致忠实度从 0.97 跌至 0.03。

### 7. 优点与亮点
*   **高效性**：证明了在单卡 H100 上即可完成 1B-8B 模型的复杂归因训练。
*   **结构化输出**：通过 JSON 格式实现了单元格级别的细粒度可验证性。
*   **创新的奖励设计**：成功将非微分的 NLI 分数引入 GRPO 框架，解决了归因质量难以量化的问题。

### 8. 不足与局限
*   **指标循环论证风险**：训练时的奖励函数和评估时的主要指标都使用了同一个 DeBERTa NLI 模型，可能存在模型“迎合”特定评分器偏好的风险，需进一步的人工评估验证。
*   **分布内评估**：实验主要集中在通用百科表格，尚未在金融、医疗等专业领域的复杂模式表格上验证泛化性。
*   **精确匹配率（EM）较低**：由于模型输出的是带推理的复杂结构，在严格的字符串匹配指标上表现一般。
*   **简洁性陷阱**：强化学习可能导致模型过度追求简洁，从而丢失部分必要的推理步骤。
