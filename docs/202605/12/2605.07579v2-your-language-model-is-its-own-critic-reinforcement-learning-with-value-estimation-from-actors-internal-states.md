## 摘要
针对大推理模型的可验证奖励强化学习（RLVR）依赖于基准估计（baseline estimation）来减少方差，但现有方法代价高昂：PPO 需要一个与策略模型规模相当的评论员（critic）模型，而 GRPO 则需要为每个提示词（prompt）进行多次采样（rollouts）以保持其经验组均值的稳定。我们提出了基于内部状态价值估计的策略优化（POISE），通过利用策略模型在前向传播过程中已经计算出的内部信号，以极低的成本获取基准。一个轻量级的探针（probe）根据提示词和生成轨迹的隐藏状态以及 token 熵统计数据来预测预期的可验证奖励，并与策略同步进行在线训练。为了在利用轨迹相关特征的同时保持梯度的无偏性，我们引入了一种交叉采样（cross-rollout）结构，通过独立采样的内部状态来预测每个采样的价值。由于 POISE 仅使用单次采样即可估计提示词价值，因此在固定的训练计算预算下，它能够实现更高的提示词多样性。这减少了梯度方差，使学习更加稳定，同时也消除了用于检测零优势提示词的采样计算开销。在 Qwen3-4B 和 DeepSeek-R1-Distill-Qwen-1.5B 上的数学推理基准测试中，POISE 在计算量更少的情况下达到了与 DAPO 相当的性能。此外，其价值估计器的表现与独立的 LLM 规模价值模型相似，并能泛化到各种可验证任务。通过利用模型自身的内部表示，POISE 实现了更稳定、更高效的策略优化。

## Abstract
Reinforcement learning with verifiable rewards (RLVR) for Large Reasoning Models hinges on baseline estimation for variance reduction, but existing approaches pay a heavy price: PPO requires a policy-model scale critic, while GRPO needs multiple rollouts per prompt to keep its empirical group mean stable. We introduce Policy Optimization with Internal State Value Estimation), which obtains a baseline at negligible cost by using the policy model's internal signals already computed during the policy forward pass. A lightweight probe predicts the expected verifiable reward from the hidden states of the prompt and generated trajectory, as well as token-entropy statistics, and is trained online alongside the policy. To preserve gradient unbiasedness despite using trajectory-conditioned features, we introduce a cross-rollout construction that predicts each rollout's value from an independent rollout's internal states. Because POISE estimates prompt value using only a single rollout, it enables higher prompt diversity for a fixed compute budget during training. This reduces gradient variance for more stable learning and also eliminates the compute overhead of sampling costs for detecting zero-advantage prompts. On Qwen3-4B and DeepSeek-R1-Distill-Qwen-1.5B across math reasoning benchmarks, POISE matches DAPO while requiring less compute. Moreover, its value estimator shows similar performance to a separate LLM-scale value model and generalizes to various verifiable tasks. By leveraging the model's own internal representations, POISE enables more stable and efficient policy optimization.

### 1. 核心问题与研究动机
在可验证奖励的强化学习（RLVR）中，为了减少梯度估计的方差并提高训练稳定性，通常需要减去一个基准值（Baseline）。
*   **现有方法的缺陷：**
    *   **PPO：** 需要一个与策略模型规模相当的独立评论员（Critic）模型，这使内存消耗翻倍，增加了优化复杂度。
    *   **GRPO：** 通过对同一提示词（Prompt）进行多次采样并取均值来作为基准，但这牺牲了提示词的多样性（固定计算预算下，每个提示词采样多，则提示词总量少），导致梯度方差增大。
*   **研究动机：** 探索是否能从策略模型前向传播中已有的内部计算信号中提取有效的基准值，从而实现“零成本”或“极低成本”的价值估计。

### 2. 方法论：POISE 核心思想与技术细节
POISE 的核心是利用策略模型自身的内部状态来训练一个轻量级的价值预测探针。
*   **核心特征提取：** 探针输入包含三个维度的信号：
    1.  **提示词特征（Prompt-level）：** 提示词最后几个 token 的隐藏状态，捕捉模型对题目难度的预判。
    2.  **轨迹特征（Trajectory-level）：** 推理结束时的隐藏状态，捕捉生成过程中的信息。
    3.  **不确定性特征：** Token 级别的熵统计量。
*   **跨样本基准构建（Cross-rollout Construction）：** 
    *   为了保证梯度的无偏性，基准必须与当前动作条件独立。
    *   POISE 为每个提示词采样两个独立响应（A 和 B）。使用响应 B 的内部状态特征来预测响应 A 的基准值，反之亦然。这种设计确保了基准值不依赖于被优化的具体动作。
*   **在线训练流程：** 
    *   探针（通常是简单的线性回归模型）与策略模型同步训练。
    *   使用一个滑动缓冲区（Sliding Buffer）存储最近的采样轨迹和奖励，不断更新探针以适应不断进化的策略。

### 6. 主要结论与发现
*   **性能匹配：** POISE 在数学推理任务上达到了与 DAPO 相当甚至更优的性能。
*   **效率提升：** 相比 GRPO 类方法，POISE 节省了约 **25%-30%** 的计算时间。
*   **稳定性：** 由于 POISE 允许在每个 batch 中包含更多样化的提示词，其梯度范数更低且更平滑，训练更稳定。
*   **内部状态的价值：** 实验证明，策略模型的内部隐藏状态确实包含了丰富的价值信息，其预测准确度甚至可以媲美一个专门训练的全规模 Critic 模型。

### 7. 优点与亮点
*   **极低开销：** 无需额外的 LLM 规模模型，仅靠一个线性层即可实现价值估计。
*   **高提示词多样性：** 突破了 GRPO 需要大组采样的限制，在相同算力下能接触更多题目，提升泛化。
*   **无偏性设计：** 巧妙的“跨样本”机制解决了利用轨迹特征带来的梯度偏差风险。
*   **通用性：** 证明了该方法不仅适用于数学，也适用于代码和指令遵循等多种可验证任务。

### 8. 不足与局限
*   **序列级基准：** 目前 POISE 提供的是序列级（Sequence-level）的基准，尚未实现更精细的 Token 级信用分配（Credit Assignment），这对于极长推理链的优化可能进一步提升空间。
*   **训练视野：** 实验主要在固定的计算预算内进行，尚未探讨在超大规模、超长时间训练下，内部状态价值估计是否会出现过拟合或失效。
*   **依赖可验证奖励：** 该方法目前主要针对有明确正确/错误反馈的任务（RLVR），对于主观性较强的偏好学习（如 DPO 场景）的适用性有待进一步研究。
