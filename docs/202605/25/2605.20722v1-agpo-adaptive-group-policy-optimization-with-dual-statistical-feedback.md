## 摘要
强化学习提升了LLM的推理能力，但PPO/GRPO通常使用固定的裁剪和解码温度，导致训练不稳定且调参繁琐。我们提出自适应组策略优化（AGPO），这是一种无评论家的GRPO改进方法，利用组级统计信息来控制更新幅度和探索强度。AGPO使用共享的探针派生统计状态驱动两个控制器：(i) 自适应裁剪，根据奖励离散度和偏度、探针投票熵、策略熵以及逐步骤KL漂移设置信任区域大小；(ii) 双向自适应温度采样，根据相对于运行基线的中心化不确定性，在基础温度附近加热或冷却解码。在九个英文和中文数学/STEM基准测试中，使用AGPO训练的Qwen2.5-14B在相同生成令牌预算下优于PPO/GRPO，在GSM8K上达到67.3%，在MATH上达到40.5%。该优势可迁移至Llama-3-8B和Gemma-2-9B，消融实验证实两个模块具有互补性。我们的实现已在https://github.com/wandugu/paper_agpo公开。

## Abstract
Reinforcement learning improves LLM reasoning, but PPO/GRPO typically use fixed clipping and decoding temperature, which makes training brittle and tuning-heavy. We propose Adaptive Group Policy Optimization (AGPO), a critic-free refinement of GRPO that uses group-level statistics to control both update magnitude and exploration. AGPO uses a shared probe-derived statistical state to drive two controllers: (i) adaptive clipping, which sets the trust-region size from reward dispersion and skewness, probe vote entropy, policy entropy, and step-wise KL drift; and (ii) bidirectional adaptive temperature sampling, which heats or cools decoding around a base temperature according to centered uncertainty relative to a running baseline. On nine English and Chinese math/STEM benchmarks, Qwen2.5-14B trained with AGPO outperforms PPO/GRPO under the same generated-token budget, reaching 67.3% on GSM8K and 40.5% on MATH. Gains transfer to Llama-3-8B and Gemma-2-9B, and ablations confirm both modules are complementary. Our implementation is publicly available at https://github.com/wandugu/paper_agpo.

## 2. 方法论：核心思想、关键技术细节
### 2.1 核心思想
- AGPO 采用**“一个探针、两个控制器”**架构：
  - **探针阶段**：在基础温度 τ_base 下采样一组（G=8）rollout，计算组级统计状态（包括奖励离散度 σ̂、偏度 |κ̃₃| 和探针熵 E_probe）。
  - **两个控制器**：
    - **自适应裁剪（adaptive clipping）**：根据探针统计量、策略熵 H(π) 和步骤 KL 漂移 D̂_step KL 动态计算 ε_adaptive，控制信任区域大小。
    - **双向自适应温度采样（ATS）**：将当前不确定性 U_t 相对于运行基线 Ū_t 中心化，得到 eU_t，从而在 τ_base 附近加热（增加探索）或冷却（增加利用）。
- 探针 rollout 仅用于统计估计，不用于梯度更新。

### 2.2 关键技术细节
- **自适应裁剪公式**（式 4）：
  ```
  ε_adaptive = clip( ε_base * H(π_old) * (1 + δ E_probe) / (1 + α σ̂ + ζ |κ̃₃| + γ D̂_step KL), ε_min, ε_max )
  ```
  使用过去步骤的 D̂_step KL（EMA 平滑）避免循环依赖。
- **自适应温度采样（ATS）**：
  - 不确定性得分 U_t = w_r σ̂ + w_e E_probe + w_k |κ̃₃|
  - 中心化后 eU_t = U_t - Ū_t（Ū_t 为 EMA 基线）
  - 温度 τ_t = clip( τ_base * (1 + λ eU_t), τ_min, τ_max )
- **损失函数**（式 7）：替换 GRPO 中的固定 ε 为 ε_adaptive，其余保持不变。
- **训练流程**：每步先探针得统计，计算 τ_t 和 ε_adaptive，再以 τ_t 采样训练 rollout，更新策略，保留 D̂_step KL 用于下一步。
- **理论保证**：给出有限时域收敛性定理（Thm 1），证明理想化代理函数在 O(1/√T) 速率下达到平稳点。
