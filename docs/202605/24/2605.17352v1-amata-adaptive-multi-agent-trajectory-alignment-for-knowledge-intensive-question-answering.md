## 摘要
尽管大型语言模型取得了显著进展，但在知识密集型问答中生成事实一致的响应仍然具有挑战性。这些困难主要源于幻觉以及大型语言模型在弥合长尾知识差距方面的局限性。为此，我们提出了AMATA，一种自适应多智能体轨迹对齐框架，该框架动态整合外部知识以提升响应的可解释性和事实依据。我们的架构利用六个专门的智能体，它们协同执行结构化动作以进行复杂问题推理。我们将与外部工具的多智能体协作形式化为轨迹偏好对齐问题，并引入了问题感知的智能体定制和智能体间偏好协调。AMATA引入了两项主要创新：(1) 轨迹内偏好学习，学习面向目标的偏好以优先考虑关键智能体；(2) 智能体间依赖学习，通过一种新颖的依赖感知直接偏好优化技术捕捉跨智能体的工具依赖关系。实验结果表明，AMATA在五个已建立的知识密集型问答基准上 consistently 优于基线方法、知识增强框架和基于大型语言模型的轨迹系统。进一步分析证明了我们的方法在减少令牌消耗方面的效率。

## Abstract
Despite substantial advances in large language models (LLMs), generating factually consistent responses for knowledge-intensive question answering remains challenging. These difficulties are primarily due to hallucinations and the limitations of LLMs in bridging long-tail knowledge gaps. To address this, we propose AMATA, an Adaptive Multi-Agent Trajectory Alignment framework that dynamically integrates external knowledge to improve response interpretability and factual grounding. Our architecture leverages six specialized agents that collaboratively perform structured actions for complex question reasoning. We formalize multi-agent collaboration with external tools as a trajectory preference alignment problem, incorporating question-aware agent customization and inter-agent preference harmonization. AMATA introduces two principal innovations: (1) Intra-Trajectory Preference Learning, which learns objective-oriented preferences to prioritize critical agents, and (2) Inter-Agent Dependency Learning, which captures cross-agent tool dependencies through a novel dependency-aware direct preference optimization technique. Empirical results show that AMATA consistently outperforms baseline approaches, knowledge-augmented frameworks, and LLM-based trajectory systems on five established knowledge-intensive QA benchmarks. Further analysis demonstrates the efficiency of our method in reducing token consumption.

- **问题**：大型语言模型（LLM）在知识密集型问答（KIQA）中面临两大挑战：**幻觉**（生成看似合理但事实错误的内容）和**长尾知识缺口**（缺乏对稀有知识的掌握）。现有缓解方法如检索增强生成（RAG）虽能引入外部知识，但存在检索不准确、推理延迟增加等缺陷。
- **现有多智能体系统的局限**：三类主流范式——**模块化训练**（智能体各自独立优化，缺乏全局协调）、**端到端训练**（统一优化但削弱智能体专长）、**全局-局部训练**（结合两者但忽略智能体间动态依赖）——均未能高效且自适应地协调工具使用顺序与依赖关系。
- **AMATA的定位**：提出自适应多智能体轨迹对齐框架，通过动态学习**轨迹内偏好**（哪些智能体对当前问题更关键）和**智能体间依赖**（如触发检索后必须执行过滤和定位），同时提升响应的事实一致性和推理效率。

#### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将多智能体协作形式化为**轨迹偏好对齐问题**，通过两阶段学习实现自适应智能体调用和依赖建模。
- **六个智能体角色**：
  | 智能体 | 功能 |
  |--------|------|
  | Intent Reconstructor (AIR) | 解析用户问题意图 |
  | Knowledge Retriever (AKR) | 从外部知识库检索相关文档 |
  | Knowledge Filter (AKF) | 过滤检索结果中的噪声 |
  | Knowledge Locator (AKL) | 精准定位支持证据文本 |
  | Response Generator (ARG) | 生成最终答案并引用来源 |
  | Answer Verifier (AAV) | 验证答案事实正确性 |
- **关键技术细节**：
  1. **轨迹内偏好学习（Intra-Trajectory Preference Learning）**：
     - 为每个问题-智能体对分配**偏好分数**（0-5），如 `<Retriever:5>` 表示该问题强烈需要检索。
     - 将这些分数作为前缀与问题拼接，对共享LLM进行**条件监督微调**（SFT），使模型学会根据问题动态调整每个智能体的重要性。
     - 损失函数：`L_Intra(Θ) = -E_{(Q,Y,P)~D_intra} log Pr(Y | P, Q; Θ)`，其中 `P` 是偏好前缀集。
  2. **智能体间依赖学习（Inter-Agent Dependency Learning）**：
     - 提出**依赖感知直接偏好优化（DA-DPO）**，超越标准DPO的单对比较，通过**列表级排序**建模智能体之间的协调关系。
     - 对每个问题，构造M个获胜样本和N个失败样本，根据**依赖分数**（衡量智能体分数组合的“良好度”）选出Top-K获胜样本，其余归为失败。
     - 利用Plackett-Luce排序模型，最大化获胜样本顺序概率，损失函数：
       ```
       L_Inter(Θ̃) = -E_{(Q,y1_w,...,yL_l)~D_inter} [ Σ_{i=1}^K log σ( -log Σ_{j=i+1}^{M+N} exp(V_β) ) ]
       ```
       其中 `V_β` 基于策略模型与参考模型的比率。
  3. **训练总损失**：`L_total = α1·L(T) + α2·L_Inter`，其中 `L(T)` 为基本轨迹语言建模损失，`α1=α2=0.5`。
- **推理流程**（算法1）：根据轨迹头令牌判断是否需要外部知识，若需要则依次执行检索、过滤、定位、生成，最后通过验证器检查正确性；若错误则重新启动轨迹并融入错误信号，最多循环3次。
