## 摘要
具有可验证奖励的强化学习有助于提升推理能力，但在所有采样展开（rollouts）均失败的困难提示词上，类 GRPO 方法往往会陷入停滞。SORT 为这些失败情况引入了一种修复更新，且无需改变展开生成过程：它从参考解中推导出计划，对比在有无该计划调节下的 token 概率，并为那些在计划调节下变得更具可预测性的 token 赋予更高的权重。这使得“全错”的提示词能够转化为具有选择性的、结构感知的学习信号，而非简单的统一模仿。在三个骨干网络和八个推理基准测试中，SORT 的表现优于 GRPO 和引导基线，且在较弱模型上的提升最为显著。

## Abstract
Reinforcement learning with verifiable rewards helps reasoning, but GRPO-style methods stall on hard prompts where all sampled rollouts fail. SORT adds a repair update for those failures without changing rollout generation: it derives a plan from the reference solution, compares token probabilities with and without that plan, and gives higher weight to tokens that become more predictable under plan conditioning. This turns all-wrong prompts into selective, structure-aware learning signals instead of uniform imitation. Across three backbones and eight reasoning benchmarks, SORT improves over GRPO and guidance baselines, with largest gains on weaker models.

### 论文总结：Selective Off-Policy Reference Tuning with Plan Guidance (SORT)

#### 1. 论文的核心问题与整体含义
论文针对**从可验证奖励中进行强化学习（RLVR）**（如 DeepSeek 提出的 GRPO 算法）中的一个关键瓶颈：**“零奖励陷阱”**。在处理困难题目时，模型生成的多个采样（rollouts）可能全部错误，导致组内相对优势（advantage）全为零，模型无法获得学习信号。
虽然这些失败样本包含最有价值的推理挑战，但简单的监督微调（SFT）会盲目模仿所有 Token（包括平庸的代数运算和格式），导致模型退化。论文提出的 **SORT** 旨在将这些失败样本转化为**具有选择性的、结构感知的学习信号**，通过“计划引导”识别并强化关键推理步骤。

#### 2. 方法论
SORT 是一种辅助的修复机制，与 GRPO 的在线采样过程并行运行。其核心思想是：**如果一个 Token 在给定“推理计划”后预测概率显著提升，那么它就承载了关键的结构化推理信息。**

*   **核心流程：**
    1.  **失败样本缓冲：** 在 GRPO 训练中，将所有采样均错误的提示词及其标准答案（Reference）存入缓冲区。
    2.  **计划提取（Plan Extraction）：** 使用当前模型从标准答案中提取一个“推理计划”（包含关键决策、引理、分类讨论等，但不作为训练目标）。
    3.  **双重评分（Dual Scoring）：** 对标准答案路径上的每个 Token 进行两次概率评估：
        *   $p_{base}$：仅基于原始提示词的预测概率。
        *   $p_{plan}$：基于“提示词 + 提取的计划”的预测概率。
    4.  **几何参考加权（Geometric Weighting）：** 计算 Token 权重 $\omega = \sqrt{p_{base} \cdot p_{plan}}$。
        *   如果 $p_{plan} \gg p_{base}$，说明该 Token 是关键推理点，赋予高权重。
        *   如果两者接近，说明是常规步骤，权重较低。
    5.  **离策更新：** 使用该权重对模型进行辅助的梯度更新，修复模型在困难题目上的推理缺陷。

#### 3. 实验设计
*   **模型底座：** Llama-3.2-3B-Instruct, Qwen2.5-7B-Instruct, Qwen3-4B-Instruct（覆盖弱、中、强三种规模）。
*   **训练数据：** OpenR1-Math-220k 的 15k 子集（源自 NuminaMath 1.5）。
*   **评测基准（Benchmarks）：**
    *   **分布内（ID）：** AIME24/25, AMC23, MATH-500, Minerva, OlympiadBench。

*   **对比方法：** SFT（基于 DeepSeek-R1 轨迹）、原生 GRPO、LUFFY（混合强模型轨迹）、ReLIFT（交替 SFT）、Scaf-GRPO（脚手架引导）。

*   **硬件：** 使用了 **8 张 NVIDIA H100 GPU**。
*   **软件框架：** 基于 `verl`（分布式 RL 框架）和 `vLLM`。
*   **训练细节：** 训练 500 个 step，batch size 为 128，每个提示词采样 8 条轨迹（Qwen3 为 4 条），学习率 $1 \times 10^{-6}$。

#### 5. 实验数量与充分性
*   **实验规模：** 论文在 3 个不同的模型底座上进行了详尽测试，涵盖了 8 个主流推理榜单。
*   **消融实验：** 针对权重指数 $\beta$（对比了 DFT 风格、纯教师风格和几何平均风格）以及不同的辅助更新变体（纯 SFT vs. SORT）进行了消融研究。
*   **充分性评价：** 实验设计较为充分且公平。所有基线方法均在相同的 15k 数据集和优化步数下重新实现，确保了对比的客观性。通过可视化 Token 显著性（Salience），直观证明了方法在识别“推理控制操作”上的有效性。

#### 6. 主要结论与发现
*   **性能领先：** SORT 在所有三个底座上均取得了平均最优性能。相比 GRPO，在 Llama-3.2-3B 上分布内提升了 2.0%，在 Qwen2.5-7B 上提升了 2.4%。
*   **弱模型受益最大：** 在 Llama-3.2-3B 这种容易出现“全错组”的弱模型上，SORT 的修复效果最显著，显著优于简单的 SFT 或轨迹注入。
*   **效率与长度：** SORT 在提升准确率的同时，避免了 LUFFY 和 ReLIFT 等方法常见的“回复长度膨胀”问题，保持了推理的简洁性。
*   **理论验证：** 理论证明了 SORT 权重能够逼近理想的“结构化 Token 权重”预测器。

#### 7. 优点
*   **选择性监督：** 不同于传统的全量模仿，SORT 能够自动识别哪些 Token 是“推理难点”，哪些是“格式废话”，实现了更精准的知识迁移。
*   **无需外部强老师：** 仅通过模型自身的“计划引导”来评分，降低了对高性能外部模型轨迹的依赖。
*   **稳定性：** 借鉴了 DFT（Dynamic Fine-Tuning）的思想，利用 $p_{base}$ 控制更新强度，避免了离策分布偏移导致的训练不稳定。

#### 8. 不足与局限
*   **依赖标准答案：** SORT 前提是必须拥有失败提示词的正确参考答案（Reference），在缺乏 Ground Truth 的开放域任务中无法直接应用。
*   **奖励形式限制：** 目前仅验证了二元（对/错）奖励场景，对于连续奖励或过程奖励（PRM）的适配性尚未探讨。
*   **计算开销：** 虽然不改变采样过程，但每个失败样本需要额外的计划生成和两次前向传播，相比原生 GRPO 增加了约 1.5 倍的训练耗时。
