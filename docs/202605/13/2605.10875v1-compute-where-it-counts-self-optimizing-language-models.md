## 摘要
高效大语言模型（LLM）推理研究主要集中在降低每个解码步骤的成本（例如，使用量化、剪枝或稀疏注意力），通常对每个生成的 token 应用统一的计算预算。在实践中，token 的难度差异很大，因此静态压缩可能会在简单步骤上过度计算，而在困难步骤上计算不足。我们研究了自回归解码的动态预算分配：在单个模型内部学习为每个 token 投入多少计算量。自优化语言模型（SOL）将冻结的 LLM 与轻量级策略网络配对，该网络读取 LLM 隐藏状态并在每个解码步骤选择离散的效率动作。这些动作可以联合控制 (i) token 级注意力稀疏性，(ii) MLP 中的结构化激活剪枝，以及 (iii) 激活量化位宽，同时保持基础模型权重不变。我们在教师强制（teacher-forced）片段上使用组相对策略优化（GRPO）来训练策略：token 序列是固定的，而我们采样多个计算调度（即仅在相同 token 路径上改变效率动作的“反事实”调度），并在相同的监督下比较它们的似然度。我们的奖励函数在语言模型质量与软惩罚之间进行权衡，后者鼓励片段平均预算使用量符合请求的目标。在不同的模型变体和计算方案中，SOL 在相同预算下比静态分配和强随机调度搜索提高了质量，为推理效率优化提供了一个互补的维度。SOL 在所有实验中都发现了更好的质量-效率帕累托前沿，并且在 MMLU 准确率上比统一预算分配策略提高了高达 7.3%。

## Abstract
Efficient LLM inference research has largely focused on reducing the cost of each decoding step (e.g., using quantization, pruning, or sparse attention), typically applying a uniform computation budget to every generated token. In practice, token difficulty varies widely, so static compression can over-compute on easy steps and under-compute on hard ones. We study dynamic budget allocation for autoregressive decoding: learning how much computation to spend per token from within a single model.   Self-Optimizing Language Models (SOL) pair a frozen LLM with a lightweight policy network that reads the LLM hidden state and selects a discrete efficiency action at each decode step. Actions can jointly control (i) token-level attention sparsity, (ii) structured activation pruning in the MLP, and (iii) activation quantization bit-width, while leaving the base model weights unchanged. We train the policy with group-relative policy optimization on teacher-forced episodes: the token sequence is fixed, while we sample multiple compute schedules (i.e., "counterfactual" schedules that vary only the efficiency actions for the same token path) and compare their likelihoods under the same supervision. Our reward trades off language-model quality against soft penalties that encourage episode-average budget usage to match a requested target. Across model variants and compute regimes, SOL improves quality at matched budget over static allocation and strong random schedule search, offering a complementary axis for inference-efficiency optimization. SOL discovers a better quality-efficiency pareto-front across all our experiments and improves MMLU accuracy by up to 7.3% over uniform budget allocation strategies.

### 论文总结：Compute Where it Counts: Self Optimizing Language Models (SOL)

#### 1. 核心问题与整体含义（研究动机和背景）
现有的 LLM 推理优化技术（如量化、剪枝、稀疏注意力）通常采用**静态分配**策略，即对生成的每一个 token 应用统一的计算预算。然而，token 的生成难度差异巨大：有些 token 具有高度局部可预测性，而有些则依赖长程上下文或复杂的中间推理。
*   **研究动机**：静态压缩会导致“简单步骤过度计算，困难步骤计算不足”。
*   **核心挑战**：token 级别的效率决策具有时间耦合性。在 $t$ 时刻过度压缩会导致 KV 缓存（KV-Cache）质量下降，这种“KV 污染”会延迟影响后续 token 的生成质量。
*   **整体含义**：本文提出了 **SOL (Self-Optimizing Language Models)** 框架，旨在通过学习一个轻量级策略网络，在保持基础模型权重冻结的情况下，为每个 token 动态分配计算资源。

#### 2. 方法论：核心思想与关键技术
SOL 的核心思想是将推理预算分配建模为一个**序列决策问题**。
*   **系统架构**：由一个冻结的基座 LLM 和一个轻量级策略网络（Policy Network，约 8.5M 参数）组成。策略网络读取 LLM 的隐藏状态、当前 token 嵌入以及预算进度特征。
*   **动作空间**：策略网络在每一步解码时选择一个离散动作，联合控制三个维度：
    1.  **Token 级注意力稀疏度 ($\kappa$)**：利用 Quest 算法选择保留的 KV 页面比例。
    2.  **结构化 MLP 激活剪枝 ($\rho$)**：根据激活幅值动态关闭 MLP 的输入通道。
    3.  **激活量化位宽 ($q$)**：在 4 到 16 bit 之间动态调整激活精度。
*   **训练算法 (GRPO)**：采用**组相对策略优化 (Group-Relative Policy Optimization)**。在“教师强制”（Teacher-forced）模式下，固定 token 序列，采样多个不同的计算调度（反事实轨迹）。
*   **奖励函数**：$R = \text{Log-Likelihood} - \text{Budget Penalty}$。通过软惩罚项（平方铰链损失）强制片段平均预算符合预设的目标值。
*   **KV 刷新机制**：为了缓解 KV 污染，每隔 $T$ 个步骤（控制时界）进行一次快速的全量计算，刷新 KV 缓存状态。

    *   **Heuristics**：基于熵（Entropy）或隐藏状态漂移（Drift）的启发式分配规则。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了从 1B 到 8B 的多个模型，动作空间从 8 个动作扩展到 1560 个动作（Fine-grained）。
*   **消融实验**：对控制时界 $T$（4, 16, 64 步）、动作粒度、预算遵循度（Budget Adherence）进行了详细分析。
*   **充分性评价**：实验设计较为充分，不仅验证了 PPL 的提升，还通过下游任务验证了实际准确率的改善。对比了强随机搜索基准，证明了策略网络学到了非平凡（non-trivial）的分配逻辑。

#### 6. 主要结论与发现
*   **帕累托前沿提升**：在所有模型规模和计算预算下，SOL 均优于静态分配策略。
*   **准确率增益**：在相同计算预算下，SOL 在 MMLU 任务上的准确率比均匀分配策略提升了高达 **7.3%**。
*   **预算可控性**：通过在输入中加入目标预算特征，单个策略网络即可在推理时根据用户需求动态调整整体压缩率。
*   **KV 污染特性**：实验发现 token 稀疏化对后续生成的负面影响（Tail Pollution）比量化和剪枝更严重，验证了动态分配的必要性。

#### 7. 优点（亮点）
*   **多维度联合优化**：首次将注意力稀疏、激活剪枝和量化位宽整合进统一的动态控制框架。
*   **训练效率高**：利用 GRPO 和反事实采样，无需昂贵的强化学习全量微调，仅需少量算力即可训练出高效的控制器。
*   **部署灵活性**：策略网络极小，推理开销几乎可以忽略不计，且支持推理时的实时预算切换。

#### 8. 不足与局限
*   **硬件适配**：虽然理论上减少了计算量，但实际延迟（Latency）提升依赖于底层算子对混合精度和动态稀疏性的支持（论文主要以 keep-rate 作为代理指标）。
*   **教师强制偏差**：训练主要在教师强制（固定路径）下进行，在自由生成（Free-running）模式下，KV 污染可能会与 token 采样误差产生更复杂的交互。
*   **层间分配缺失**：目前的策略是跨层统一应用相同的动作，尚未实现更细粒度的“按层分配”计算资源。
