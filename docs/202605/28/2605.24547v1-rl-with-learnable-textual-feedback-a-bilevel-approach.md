## 摘要
基于可验证奖励的强化学习可以改进大语言模型的推理能力，但当终端奖励稀疏时，样本效率仍然低下。这推动了关于文本反馈强化学习的日益增长的研究方向，即批评模型生成自然语言反馈来指导推理模型（行动者），通过更丰富的学习信号增强标量奖励。然而，现有方法通常将反馈视为固定或辅助性的，这忽略了一个关键性质：反馈不仅应该正确，而且当在上下文中提供时，应该能改善策略（行动者模型）。这推动了可学习文本反馈的强化学习范式。然而，反馈的可学习性和有用性取决于策略从中学习的能力，这使得具有可学习反馈的强化学习本质上是一个双层问题。我们将这种耦合形式化为Stackelberg双层规划，并推导出双层自然语言行动者-批评者（Bi-NAC），它联合训练批评者以生成能改善奖励的反馈，以及行动者以利用这种反馈。在MATH-500、MBPP和GPQA上，Bi-NAC相比于强化学习和固定批评者基线提高了样本和参数效率：我们的2B模型优于3B GRPO基线，在MATH-500上达到46.6%对比41.4%；而我们的6B模型在GPQA上超越7B GRPO基线，达到49.3%对比43.6%。

## Abstract
Reinforcement learning with verifiable rewards can improve LLM reasoning, but learning remains sample-inefficient when terminal rewards are sparse. This has motivated a growing line of work on RL with textual feedback, where a critic model generates natural language feedback to guide a reasoning model (the actor), augmenting scalar rewards with richer learning signals. However, existing methods typically treat feedback as fixed or auxiliary, which misses a key property: feedback should not merely be correct, but should improve the policy (actor model) when provided in context. This motivates a paradigm of learnable textual feedback for RL. Yet the learnability and usefulness of feedback depend on the policy's ability to learn from it, making RL with learnable feedback an inherently bilevel problem. We formalize this coupling as a Stackelberg bilevel program and derive Bilevel Natural Language Actor-Critic (Bi-NAC), which jointly trains a critic to generate reward-improving feedback and an actor to exploit it. Across MATH-500, MBPP, and GPQA, Bi-NAC improves sample and parameter efficiency over RL and fixed-critic baselines: our 2B model outperforms the 3B GRPO baseline, achieving 46.6% versus 41.4% on MATH-500, while our 6B model surpasses the 7B GRPO baseline, achieving 49.3% versus 43.6% on GPQA.

## 1. 核心问题与整体含义
- **研究动机**：基于可验证奖励的强化学习（如 GRPO）能提升大语言模型的推理能力，但稀疏的终端二元奖励导致**优势塌缩**（advantage collapse）——当一个组内所有采样均错误时，优势归零，梯度更新消失。实验表明在 MBPP 上优势塌缩频率高达 74%，奖励塌缩频率达 83%。
- **现有方法的不足**：当前使用文本反馈（critic 生成自然语言指导）的方法将反馈视为固定或辅助组件，未针对行动者的下游改进进行优化。即使反馈本身正确，也可能无法帮助行动者获得更高奖励（即反馈不可操作）。
- **核心洞察**：反馈的可学习性与有用性依赖于行动者从中学习的能力，因此反馈生成与策略学习是**天然耦合的双层问题**。需要将二者联合建模，实现可学习的文本反馈。

## 2. 方法论
- **核心思想**：将反馈生成与策略学习形式化为 **Stackelberg 双层规划**：
  - **下层**（actor）：在给定批评者生成的反馈条件下，最大化验证奖励。
  - **上层**（critic）：通过行动者的学习动态优化自身参数，以最大化行动者适应反馈后的最终奖励。
- **关键技术**：
  - **Bi-NAC 框架**：联合训练行动者 LLM（参数 θ）和批评者 LLM（参数 φ）。具体流程：行动者生成初始回答 y0 → 批评者生成反馈 z → 行动者根据反馈生成修正回答 y1，获得奖励 R。
  - **拉格朗日形式**：将双层约束转化为无约束优化 ℒ(φ,θ,λ) = U(φ,θ) + λ[L(φ,θ) - L(φ,θ*(φ))]，其中 θ*(φ) 为给定 φ 下的最优行动者。
  - **梯度推导**：
    - 批评者梯度（式 6）：∇φℒ = (1+λ)Eτ~πθ,φ[∇φlogπHφ(z|x,y0)R] - λEτ*~πθ*,φ[∇φlogπHφ(z|x,y0)R]，鼓励批评者生成能提升当前行动者奖励的反馈，同时被“最优”行动者的表现所正则。
    - 行动者梯度（式 7）：∇θℒ = (1+λ)E[ (∇θlogπLθ(y0|x) + ∇θlogπLθ(y1|x,y0,z)) R ]，为标准策略梯度形式。
  - **算法实现**（Algorithm 1）：交替更新行动者（基于当前反馈最大化奖励）和批评者（基于行动者的样本和“最优”行动者的样本），其中最优行动者通过历史检查点近似。
- **关键特性**：批评者的优化显式依赖于行动者的学习动态，从而产生**对齐的、可操作的**反馈。
