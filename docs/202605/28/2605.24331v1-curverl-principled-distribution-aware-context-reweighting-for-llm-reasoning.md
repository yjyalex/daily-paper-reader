## 摘要
上下文或提示级别的重加权已成为强化学习与验证奖励（RLVR）中提升大语言模型推理能力的核心算法杠杆，然而确定最优加权的原则仍然不甚明了。我们通过将提示重加权定义为通过率函数空间中效用泛函的泛函导数来填补这一空白，从而得到一个统一的最优性框架，该框架涵盖了包括REINFORCE和GRPO在内的现有方案。基于这一最优性框架，我们提出了一种分布感知的提示重加权方法，称为CurveRL，该方法基于分位数坐标变换，其中分配给每个提示的权重不取决于通过率的绝对值，而是取决于其在学习动态中通过率分布结构中的排名和密度。跨多个基准的大量实验表明，我们提出的CurveRL始终优于GRPO和其他RLVR基线。我们的研究将上下文分布控制确定为分析和设计提示重加权RLVR算法的一个原则性轴心。代码发布在https://github.com/zhyzmath/CurveRL。

## Abstract
Context or prompt-level reweighting has emerged as a central algorithmic lever in Reinforcement Learning with Verified Rewards (RLVR) for improving the reasoning capability of large language models, yet the principle determining what constitutes an optimal weighting remains poorly understood. We address this gap by formulating prompt reweighting as a functional derivative of a utility functional defined in the pass-rate function space, yielding a unified optimality framework that accommodates existing schemes, including REINFORCE and GRPO. Building on this optimality framework, we propose a distribution-aware prompt reweighting approach, called CurveRL, based on a quantile coordinate transform, in which the weight assigned to each prompt depends not on the absolute value of pass rates but on its rank and density to reflect the distributional structure of the pass rates in the learning dynamics. Extensive experiments across multiple benchmarks demonstrate that our proposed CurveRL consistently outperforms GRPO and other RLVR baselines. Our study identifies context-distribution control as a principled axis for analyzing and designing prompt-reweighted RLVR algorithms. The code is released in https://github.com/zhyzmath/CurveRL.

### 2. 论文提出的方法论

- **核心思想**：将提示重加权形式化为**通过率函数空间中的泛函导数**。定义效用泛函 \(U_\theta\) 依赖于通过率函数 \(p_\theta(\cdot)\)，最优权重 \(w_\theta^*(x)\) 等于该泛函在点 \(x\) 处的泛函导数 \(\frac{\delta U_\theta}{\delta p_\theta}(x)\)。点式效用（如REINFORCE、GRPO、MaxRL）只是泛函导数的退化情况。
- **关键技术细节**：
  - **分布感知效用泛函**：不是使用点式变换 \(g(p_\theta(x))\)，而是引入参考分布 \(F_{\text{ref}}\)（由滞后策略的通过率估计），构建效用 \(U_\theta(F_{\text{ref}}) = \mathbb{E}_{x\sim d_0}[\psi(F_{\text{ref}}(p_\theta(x)))]\)，其中 \(\psi\) 为失真函数（本文取 \(\psi(u) = \log u\)）。
  - **权重推导**：通过泛函导数得到梯度：\(\nabla_\theta U_\theta = \mathbb{E}_{x\sim d_0}\left[\frac{f_{\text{ref}}(p_\theta(x))}{F_{\text{ref}}(p_\theta(x))} \nabla_\theta p_\theta(x)\right]\)。权重由通过率在参考分布中的排名（通过CDF）和密度决定，而非绝对值。
  - **算法流程（CurveRL）**：
    1. 维护一个长度为 \(t_0 \times |B|\) 的滑动窗口 \(W\)，存储最近 \(t_0\) 步的通过率。
    2. 对每个输入 \(x\)，采样 \(N\) 个响应，计算经验通过率 \(\hat{p}\)。
    3. 通过柱状图估计 \(\hat{f}_{\text{ref}}(\hat{p})\) 和 \(\hat{F}_{\text{ref}}(\hat{p})\)，计算权重 \(w_t(\hat{p}) = \hat{f}_{\text{ref}}(\hat{p}) / \hat{F}_{\text{ref}}(\hat{p})\)。
    4. 使用 \(\hat{p}\) 作为基线，计算梯度 \(\hat{g}(x) = \frac{1}{N}\sum_{i=1}^N w_t(\hat{p})(r_i - \hat{p})\nabla_\theta \log \pi_\theta(y_i|x)\)。
    5. 更新窗口，仅保留 \(\hat{p} \in (0,1)\) 的样本。

- **理论分析**：
  - 定理1：任何点式权重 \(w(p)\) 均可视为某种先验分布 \(F_{\text{ref}}\) 下的分布感知权重（即 \(w(p)=f_{\text{ref}}(p)/F_{\text{ref}}(p)\)）。
  - 命题3：分布感知权重具有**单调校准不变性**（对通过率的单调变换不影响梯度形式），而点式权重不具备此性质。

- **性能优势**：CurveRL在pass@64上平均超越最强基线MaxRL：+5.9%（1.7B）、+9.7%（4B）；同时在pass@1上也保持或略优，即**改善了pass@1与pass@k的Pareto前沿**。

- **扩大可解边界**：CurveRL减少了“不可解”提示的比例，将更多提示推入中等难度区间（通过率>0.5），使多数投票（majority voting）更加可靠，从而提升了无奖励引导的测试时扩展能力。
- **训练稳定性**：CurveRL在训练过程中保持了更高的策略熵和平滑的梯度范数，训练动态更稳定。

### 7. 优点

1. **理论严谨**：首次将提示重加权形式化为泛函导数，建立统一最优性框架，统一解释现有算法并揭示了其局限性。

3. **数据驱动**：通过滑动窗口自适应估计参考分布，无需预先设定通过率与权重的解析关系。

### 8. 不足与局限

1. **点式与分布感知集成的尝试未成功**：论文尝试了加性/乘性整合两者，但无改进，可能由于两种效用几何冲突，未来需进一步研究。
