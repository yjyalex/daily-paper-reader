## 摘要
针对大推理模型（Large Reasoning Models）的带可验证奖励的强化学习（RLVR）依赖于基准估计（baseline estimation）来减少方差，但现有方法代价高昂：PPO 需要一个与策略模型规模相当的评论家（critic）模型，而 GRPO 则需要针对每个提示词（prompt）进行多次采样（rollouts）以保持其经验组均值的稳定。我们提出了基于内部状态价值估计的策略优化（POISE），它通过利用策略模型在前向传播过程中已经计算出的内部信号，以极低的成本获取基准。一个轻量级的探针（probe）根据提示词和生成轨迹的隐藏状态以及 token 熵统计数据来预测预期的可验证奖励，并与策略模型同步进行在线训练。为了在利用轨迹相关特征的同时保持梯度的无偏性，我们引入了一种交叉采样（cross-rollout）结构，通过独立采样的内部状态来预测每个采样的价值。由于 POISE 仅使用单次采样即可估计提示词价值，因此在固定的训练计算预算下，它能够实现更高的提示词多样性。这减少了梯度方差，使学习更加稳定，同时也消除了用于检测零优势提示词的采样计算开销。在 Qwen3-4B 和 DeepSeek-R1-Distill-Qwen-1.5B 上的数学推理基准测试中，POISE 在计算量更少的情况下达到了与 DAPO 相当的性能。此外，其价值估计器的表现与独立的 LLM 规模价值模型相似，并能泛化到各种可验证任务。通过利用模型自身的内部表示，POISE 实现了更稳定、更高效的策略优化。

## Abstract
Reinforcement learning with verifiable rewards (RLVR) for Large Reasoning Models hinges on baseline estimation for variance reduction, but existing approaches pay a heavy price: PPO requires a policy-model scale critic, while GRPO needs multiple rollouts per prompt to keep its empirical group mean stable. We introduce Policy Optimization with Internal State Value Estimation), which obtains a baseline at negligible cost by using the policy model's internal signals already computed during the policy forward pass. A lightweight probe predicts the expected verifiable reward from the hidden states of the prompt and generated trajectory, as well as token-entropy statistics, and is trained online alongside the policy. To preserve gradient unbiasedness despite using trajectory-conditioned features, we introduce a cross-rollout construction that predicts each rollout's value from an independent rollout's internal states. Because POISE estimates prompt value using only a single rollout, it enables higher prompt diversity for a fixed compute budget during training. This reduces gradient variance for more stable learning and also eliminates the compute overhead of sampling costs for detecting zero-advantage prompts. On Qwen3-4B and DeepSeek-R1-Distill-Qwen-1.5B across math reasoning benchmarks, POISE matches DAPO while requiring less compute. Moreover, its value estimator shows similar performance to a separate LLM-scale value model and generalizes to various verifiable tasks. By leveraging the model's own internal representations, POISE enables more stable and efficient policy optimization.

### 论文结构化深入总结：POISE (Policy Optimization with Internal State Value Estimation)

#### 1. 核心问题与整体含义（研究动机和背景）
在大型推理模型（LRM）的强化学习（RLVR）中，为了减少梯度估计的方差并提高训练稳定性，必须引入**基准值（Baseline）**。现有的主流方法存在显著缺陷：
*   **PPO (Proximal Policy Optimization)**：需要一个与策略模型规模相当的独立评论家（Critic）模型，这导致显存占用翻倍，计算复杂度大幅增加。
*   **GRPO (Group Relative Policy Optimization)**：通过对同一提示词（Prompt）进行多次采样并取均值来作为基准。这虽然省去了 Critic 模型，但在固定计算预算下降低了提示词的多样性（Batch 内不同 Prompt 减少），导致梯度方差增大，且在处理“全对”或“全错”的无效提示词时浪费大量算力。

**研究动机**：能否不增加额外大模型，也不依赖多样本采样，直接从策略模型自身的计算过程中提取出有效的基准信号？

---

#### 2. 方法论：POISE 核心思想与技术细节
POISE 的核心思想是**利用策略模型（Actor）在前向传播中产生的内部隐藏状态和统计信息来估计价值（Value）**。

*   **特征提取（Internal State Features）**：
    *   **提示词特征**：提取 Prompt 最后 10 个 token 的隐藏状态均值，捕捉模型对题目难度的预判。
    *   **推理特征**：提取生成轨迹最后 10 个 token 的隐藏状态均值，捕捉推理过程中的质量信号。
    *   **不确定性特征**：提取 Token 级别的熵统计量（Entropy）。
*   **轻量级探针（Lightweight Probe）**：使用一个极其简单的线性回归模型（Ridge Regression）作为价值估计器，输入上述特征，预测预期的可验证奖励。
*   **交叉采样构造（Cross-Rollout Construction）**：
    *   为了保证梯度的**无偏性**，基准值必须与当前动作条件独立。
    *   POISE 对每个 Prompt 采样两个独立响应（$y^{(1)}, y^{(2)}$）。在更新 $y^{(1)}$ 时，使用 $y^{(2)}$ 的内部状态特征来预测基准值，反之亦然。这种设计确保了基准值不包含当前响应的特定信息。
*   **在线训练**：探针与策略模型同步训练，利用一个滑动窗口缓冲区（Buffer）存储最近的轨迹特征和奖励，确保价值估计器能跟上策略的演进。

#### 7. 优点（亮点）
*   **极高的计算效率**：复用了前向传播的中间产物，几乎以“零成本”获取了高质量的基准值。
*   **巧妙的无偏设计**：通过交叉采样（Cross-Rollout）解决了轨迹相关特征引入的梯度偏差问题。
*   **即插即用**：不需要复杂的超参数调优，线性探针在不同模型和任务上表现稳健。
*   **解决了 GRPO 的痛点**：消除了为了估计组均值而必须进行的冗余采样，提高了数据利用率。

#### 8. 不足与局限
*   **序列级信用分配**：目前 POISE 提供的是序列级（Sequence-level）的优势信号，对于极长推理链，可能不如 Token 级的信用分配（Credit Assignment）精确。
*   **长期训练表现未知**：实验主要在固定计算预算内进行，尚未观察在超大规模、超长时间训练下内部状态信号是否会发生退化或被模型“作弊”绕过。
*   **依赖可验证奖励**：虽然论文提到泛化，但核心仍依赖于有明确 Ground Truth 的任务（RLVR 场景），在主观性较强的任务（如创意写作）中效果待证。
