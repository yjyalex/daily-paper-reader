## 摘要
工具集成推理 (TIR) 难以扩展到小语言模型，原因在于长程工具交互的不稳定性以及有限的模型容量。虽然像群体相对策略优化 (GRPO) 这样的强化学习方法仅提供稀疏的结果级奖励，但最近在线策略蒸馏 (OPD) 因其能在学生生成的轨迹上提供来自教师模型的密集 Token 级监督而受到关注。然而，我们的实验表明，将 OPD 应用于 TIR 会导致一种关键的失效模式：错误的工具调用往往会在后续推理步骤中产生级联效应，逐渐放大学生与教师之间的分歧，使得教师的 Token 级监督变得越来越不可靠。为解决这一问题，我们提出了 SOD，一种面向小语言模型智能体的逐步在线策略蒸馏框架，该框架根据步骤级分歧自适应地重新调整每一步的蒸馏强度。因此，SOD 能够减弱高分歧区域中潜在的误导性教师信号，同时在对齐良好的状态下保留密集的指导。在具有挑战性的数学、科学和代码基准测试上的实验表明，SOD 相比次优基准模型实现了高达 20.86% 的提升。值得注意的是，我们的 0.6B 学生模型在 AIME 2025 上达到了 26.13% 的准确率，证明了智能体推理能力向轻量级模型的有效迁移。我们的代码可在 https://github.com/YoungZ365/SOD 获取。

## Abstract
Tool-integrated reasoning (TIR) is difficult to scale to small language models due to instability in long-horizon tool interactions and limited model capacity. While reinforcement learning methods like group relative policy optimization provide only sparse outcome-level rewards. Recently, on-policy distillation (OPD) has gained popularity by supplying dense token-level supervision from a teacher on student-generated trajectories. However, our experiments indicate that applying OPD to TIR leads to a critical failure mode: erroneous tool calls tend to cascade across subsequent reasoning steps, progressively amplifying student-teacher divergence and rendering the teacher's token-level supervision increasingly unreliable. To address this, we propose SOD, a step-wise on-policy distillation framework for small language model agents, which adaptively reweights distillation strength at each step based on step-level divergence. Therefore, SOD can attenuate potentially misleading teacher signals in high-divergence regions while preserving dense guidance in well-aligned states. Experiments on challenging math, science, and code benchmarks show that SOD achieves up to 20.86% improvement over the second-best baseline. Notably, our 0.6B student achieves 26.13% on AIME 2025, demonstrating effective transfer of agentic reasoning to lightweight models. Our code is available at https://github.com/YoungZ365/SOD.

### 1. 核心问题与研究动机
*   **核心问题**：如何有效地将复杂的“工具集成推理”（TIR）能力从大模型迁移到小语言模型（SLM）上。
*   **研究背景**：
    *   SLM 在处理长程工具交互时极不稳定，且容量有限。
    *   现有的强化学习（如 GRPO）仅提供稀疏的结果级奖励，导致小模型探索效率低下。
    *   现有的在线策略蒸馏（OPD）虽然提供密集的 Token 级监督，但在 TIR 场景下存在**“状态漂移”**问题：一旦学生模型在某一步调用工具出错，后续上下文就会被污染，此时教师模型的 Token 级指导会变得不可靠甚至产生误导，导致训练崩溃。

### 2. 方法论
*   **核心思想**：提出**逐步在线策略蒸馏（SOD）**，通过在每个推理步骤（Step）动态评估学生与教师的分布差异，自适应地调整蒸馏权重。
*   **关键技术细节**：
    *   **步骤级分歧评分 ($d_k$)**：计算第 $k$ 步中学生模型与教师模型在 Token 预测对数概率上的平均绝对差值。
    *   **自适应权重 ($w_k$)**：
        *   初始步权重 $w_1=1$。
        *   后续步权重基于相邻步的分歧比率计算：若分歧增加（$d_{u+1} > d_u$），则减小权重以抑制误导信号；若分歧减小（$d_{u+1} < d_u$，即模型自我修复），则增加权重以恢复指导。
        *   设置权重上限（$1+\delta$）以保证稳定性。
    *   **训练目标**：结合了 GRPO 的结果级奖励（引导探索）和 SOD 的逐步 Token 级蒸馏（提供密集且可靠的指导）。

### 3. 实验设计
*   **数据集**：

*   **基准测试（Benchmarks）**：AIME 2024/2025（数学）、GPQA-Diamond（科学）、LiveCodeBench（代码）。
*   **对比方法**：Vanilla（原始模型）、SFT、GRPO、标准 OPD、以及两种在线自蒸馏变体（OPSD）。
*   **模型配置**：学生模型为 Qwen3-0.6B 和 1.7B；教师模型为 Qwen3-4B 和 14B。

### 6. 主要结论与发现
*   **性能卓越**：SOD 在所有基准上均显著优于所有基线，比次优的 OPD 提升高达 20.86%。
*   **小模型突破**：0.6B 模型在 AIME 2025 上达到 26.13% 的准确率，是目前已知首个在该难度基准上达到此水平的 1B 以下模型。
*   **稳定性**：SOD 有效解决了 GRPO 的“熵崩溃”问题（即模型停止探索，陷入重复输出）和 OPD 的“训练不稳定”问题。
*   **三种模式**：识别出 SOD 在训练中表现出的三种模式：稳定模式、错误抑制模式、以及关键的**恢复模式**（允许模型在暂时偏离后重新获得教师指导）。

### 7. 优点与亮点
*   **针对性强**：精准捕捉到了 TIR 任务中“工具调用导致状态不连续”这一痛点。
*   **自适应性**：权重机制不仅能抑制错误，还能识别模型的“自我修复”行为并恢复蒸馏强度。
*   **极高效率**：在不增加推理或训练复杂度的前提下，大幅提升了超小规模模型的逻辑推理上限。

### 8. 不足与局限
*   **工具环境单一**：实验主要集中在 Python 代码解释器环境，尚未在更复杂的 Web 浏览或多 API 调用场景中验证。
*   **模型家族局限**：所有实验均基于 Qwen3 系列模型，虽然该系列具有代表性，但跨模型家族（如 Llama 或 Gemma）的泛化性有待进一步验证。
*   **依赖教师质量**：虽然 SOD 能缓解分歧，但其性能上限仍受限于教师模型在特定任务上的表现。
