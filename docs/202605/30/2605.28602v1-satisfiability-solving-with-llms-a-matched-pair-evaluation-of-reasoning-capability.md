## 摘要
大语言模型（LLMs）越来越多地被用于隐含地归结为布尔可满足性（SAT）的任务，但它们在SAT上的推理能力仍不清楚。我们对LLMs在2-SAT和3-SAT上进行了系统研究，并附带两个规范归约——顶点覆盖和离散3D装箱——以探测表示不变的推理。我们首先使用传统指标评估模型，包括准确率、精确率、召回率和F1，以及SAT相变设置。我们发现这些指标可能具有误导性：许多模型通过过度预测可满足公式而获得高分，未能复现3-SAT阈值附近的经典易-难-易特征，并且随着变量数量的增加而急剧下降。为了解决这个问题，我们引入了一种基于最小差异可满足和不可满足实例的配对公式协议，以及准确区分率（ADR），该指标要求每个实例对中的两个成员都被正确分类。ADR将面向推理的模型与启发式模型区分开来，并与见证有效性相关联。在CNF之外，我们通过将CNF转换为顶点覆盖和将3-SAT转换为离散3D装箱来测试跨表示一致性。大多数模型在超过80%的实例上，其在CNF以及相应的图或装箱实例上的决策一致，表明跨表示存在稳定的决策规则。总体而言，我们的结果表明，SAT是LLM推理的一个保守探针，而使用ADR的配对评估比传统指标提供了更忠实且对表示更鲁棒的评估。

## Abstract
Large language models (LLMs) are increasingly used for tasks that implicitly reduce to Boolean satisfiability (SAT), yet their reasoning ability on SAT remains unclear. We present a systematic study of LLMs on 2-SAT and 3-SAT, together with two canonical reductions, Vertex Cover and discrete 3D packing, to probe representation-invariant reasoning. We first evaluate models using conventional metrics, including accuracy, precision, recall, and F1, as well as the SAT phase-transition setting. We find that these metrics can be misleading: many models obtain high scores by over-predicting satisfiable formulas, fail to reproduce the classical easy-hard-easy signature around the 3-SAT threshold, and degrade sharply as the number of variables grows.   To address this problem, we introduce a paired-formula protocol based on minimally different satisfiable and unsatisfiable instances, together with Accurate Differentiation Rate (ADR), which requires both members of each pair to be classified correctly. ADR separates reasoning-oriented models from heuristic ones and correlates with witness validity. Beyond CNF, we test cross-representation consistency by converting CNF to Vertex Cover and 3-SAT to discrete 3D packing. Model decisions on CNF and on the corresponding graph or packing instances agree for most models on more than 80 percent of instances, suggesting stable decision rules across representations. Overall, our results show that SAT is a conservative probe for LLM reasoning, and that paired evaluation with ADR provides a more faithful and representation-robust assessment than conventional metrics.

### 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：通过构建“最小差异可满足/不可满足实例对”（paired formulation）来消除类别不平衡和表面线索的干扰，并用**准确区分率 (ADR)** 衡量模型是否同时正确分类一对中的两个实例。此外，将 CNF 公式通过规范归约转换为顶点覆盖和离散 3D 装箱问题，测试跨表示的推理一致性。
- **关键技术细节**：
    - **配对实例生成**：对每个由 solver 验证为 UNSAT 的 3-CNF 公式，通过三次编辑尝试（翻转文字极性、替换变量、删除子句）快速生成对应的 SAT 实例（经 MiniSat 验证），保持变量集固定，α 变化不超过 1/N。
    - **ADR 定义**：  
      `ADR = (1/M) * Σ_i I( ŷ(SAT_i) = SAT ∧ ŷ(UNSAT_i) = UNSAT )`  
      即要求模型对同一对的两个实例都给出正确决策。ADR 严格优于传统指标：它 ≤ min(recall_SAT, recall_UNSAT) ≤ Acc = (recall_SAT + recall_UNSAT)/2。
    - **跨表示测试**：将 CNF 实例按标准 Karp 归约转换为 Vertex Cover 实例（有界大小 k）和离散 3D 装箱实例（可满足性镜像），要求模型输出“YES/NO”决策，并论证 ADR 仍适用。
- **公式算法流程（文字说明）**：
    1. 生成随机 3-CNF 公式，用 MiniSat 标记为 SAT/UNSAT。
    2. 对每个 UNSAT 公式，通过编辑操作创建配对 SAT 公式（由 MiniSat 验证）。
    3. 将同一对的两个公式分别输入 LLM，收集其 SAT/UNSAT 分类决策。
    4. 计算传统指标（Acc、Pre、Rec、F1、MCC）和 ADR。
    5. 对配对 CNF 进行 Vertex Cover 和 3D Packing 归约，重复上述评估，比较跨表示的一致性。

### 6. 论文的主要结论与发现
- **传统指标具有误导性**：在相变实验中，多数 LLM 表现出强烈的 SAT 偏向，导致准确率、精确率、召回率、F1 等指标在低 α 区看似很高，但模型实际上并未复现经典的易-难-易相变模式。MCC 在类别倾斜时频繁未定义。
- **ADR 更可靠地反映推理能力**：ADR 随 N 增加平稳下降，能清晰区分推理型模型（GPT-5、o1 等）和通用型模型，后者在 N≥25 时 ADR 常低于 0.2（接近随机）。
- **跨表示一致性高**：在 CNF、顶点覆盖、3D 装箱之间，LLM 的决策一致性超过 80%。当不一致时，CNF 上的决定略优于 VC，与 3D 装箱决定正确性相近。
- **规模仍是主要瓶颈**：即使最佳推理模型（GPT-5）在 N=25 时 ADR=0.79，但在 N=50 时 ADR 降至 0.35（且完成率仅 20%）。无模型能在较大 N（>50）下稳健求解 3-SAT。
- **SAT 是推理能力的保守探针**：即 LLM 在 SAT 上的性能（尤其是在配对评估下）可作为其在一系列 NP 约化任务上推理能力的上界。

### 7. 优点：方法或实验设计上的亮点
- **提出 ADR 新指标**：简洁、直观、无需额外参数，直接测量模型区分最小差异实例的能力，避免了类别不平衡和单向偏向的干扰。
- **配对实例生成可复现**：编辑操作直观，能由 miniSat 验证，保证了 SAT/UNSAT 标签的准确性和实例的结构相似性。
- **跨表示一致性验证**：通过规范约化将 SAT 决策推广到图问题和装箱问题，验证了模型决策的稳定性，增加了结论的泛化性。
- **实验设计紧扣理论**：基于 SAT 相变理论设置 α 区间，使用 CDCL 求解器作为理性基准，为 LLM 提供可比的难度参考。
- **公开数据集与代码**：提供了完整的 Zenodo 存档和 GitHub 仓库，便于复现。

### 8. 不足与局限
- **实验规模较小**：每配置下仅 70 对实例，且部分模型因 API 预算或超时未覆盖所有 N，统计效力有限，可能遗漏更细微的差异。
- **仅限于随机 SAT 实例**：未测试真实世界编码的 SAT（如来自软件验证、规划、电路设计的实例）。随机实例的结构较为简单，可能无法充分暴露 LLM 在复杂逻辑模式下的缺点。
- **配对生成依赖编辑规则**：虽然经过 solver 验证，但三次编辑不一定能保证生成“最小差异”，且 SAT/UNSAT 对之间的差异可能并非单一步骤导致，存在引入额外线索的风险。
- **ADR 不衡量端到端求解质量**：它只关注决策正确性，不评估求解效率、内存使用、证明复杂度等实用维度。高 ADR 不一定意味着实用求解能力强。
- **模型内部状态不可控**：LLM 报告的“分支数/冲突数”与 CDCL 数值严重不一致，无法作为可靠的过程指标。
- **标签准确性依赖外部 solver**：如果 solver 本身有误（可能性小），将影响所有指标。
- **外部推广性有限**：虽然作者论证了 SAT 是保守探针，但仅测试了两种归约（顶点覆盖、3D 装箱）。其他类型归约（如图着色、独立集）未测试，结论可能无法完全推广。
