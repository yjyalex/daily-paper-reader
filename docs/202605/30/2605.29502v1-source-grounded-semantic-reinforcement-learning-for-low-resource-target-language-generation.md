## 摘要
低资源目标语言生成通常受限于稀缺的平行数据，而高资源源语言单语数据虽然丰富，但难以通过标准监督微调加以利用。我们提出源语言接地语义强化学习（SG-SRL），这是一种资源利用框架，可将源语言单语数据转换为用于目标语言生成的跨语言语义监督。SG-SRL使用跨语言语义奖励模型（由跨语言重排序器实例化，该重排序器对源输入与目标语言生成之间的语义相关性进行评分）在源语言数据上执行无参考强化学习（RL）。虽然这会导致严重的基于冗长性的奖励作弊，但一种使用小型平行语料库的轻量级恢复阶段可在保留语义增益的同时恢复流畅性、简洁性和任务格式。在中文到泰语生成上的实验表明，SG-SRL相比冷启动SFT改善了语义接地和事实覆盖率。对长文本迁移和基于藏语嵌入奖励的进一步分析阐明了SG-SRL的泛化行为，并表明在现实的低资源语言环境中，基于编码器的语义奖励可以替代基于LLM的重排序器。

## Abstract
Low-resource target-language generation is often limited by scarce parallel data, while high-resource source-language monolingual data is abundant but difficult to use with standard supervised fine-tuning. We propose Source-Grounded Semantic Reinforcement Learning (SG-SRL), a resource-utilization framework that converts source-language monolingual data into cross-lingual semantic supervision for target-language generation. SG-SRL performs reference-free reinforcement learning (RL) on source-language data using a cross-lingual semantic reward model, instantiated by a cross-lingual reranker that scores the semantic relevance between the source input and the target-language generation. While this induces severe verbosity-based reward hacking, a lightweight recovery stage using a small parallel corpus restores fluency, conciseness, and task format while preserving the semantic gains. Experiments on Chinese-to-Thai generation show that SG-SRL improves semantic grounding and factual coverage over cold-start SFT. Additional analyses on long-form transfer and Tibetan embedding-based rewards clarify the generalization behavior of SG-SRL and show that an encoder-based semantic reward can substitute for an LLM-based reranker in a realistic low-resource language setting.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：低资源目标语言生成（如泰语、藏语）面临平行数据稀缺的严重瓶颈。传统方法依赖大规模目标语言文本或平行语料进行监督微调，但收集成本高昂。而高资源源语言（如中文）的单语数据丰富，却无法直接用于监督训练。
- **动机**：探索如何将丰富的源语言单语数据转化为有用的语义监督信号，提升低资源目标语言生成的质量，无需依赖目标语言参考答案。
- **贡献**：提出**源语言接地语义强化学习（Source-Grounded Semantic Reinforcement Learning, SG-SRL）**框架，通过跨语言语义奖励实现无参考强化学习，并设计轻量级恢复阶段抑制基于冗长性的奖励黑客问题。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：采用“训练-强化-恢复”三阶段解耦设计：
  1. **目标形式初始化**：在小规模平行语料 $D_{\text{para}}$ 上进行监督微调（SFT），让模型学习基本目标语言形式、任务格式和局部生成风格。
  2. **源语言接地语义强化学习**：在大规模源语言单语数据 $D_{\text{src}}$ 上进行无参考RL。使用**跨语言重排序器**作为语义奖励模型：将源输入视为查询，目标生成视为候选文档，重排序器输出“是/否”相关性概率作为奖励。为抑制退化行为，设计三种奖励护卫：
     - **语言门**：强制输出为目标语言，否则奖励为0。
     - **批相对长度惩罚**：当输出长度超过批中位数的2.5倍时线性增加惩罚。
     - **重复惩罚**：基于重复4-gram比例，超过0.15时惩罚。
     最终奖励 $r_i = \max(\text{rerank\_score} - \lambda_{\text{len}}\cdot p_{\text{len}} - \lambda_{\text{rep}}\cdot p_{\text{rep}}, 0)$，并用GRPO进行策略优化。
  3. **目标形式恢复**：在语义RL后，复用 $D_{\text{para}}$ 进行1轮轻量SFT，恢复流利性、简洁性和任务格式，同时保留语义增益。

- **关键技术细节**：
  - 奖励模型：生成式yes/no重排序器，计算 $\text{rerank\_score} = \frac{\exp(z_{\text{yes}})}{\exp(z_{\text{yes}})+\exp(z_{\text{no}})}$。
  - 策略优化算法：GRPO（Group Relative Policy Optimization），每组内采样多个候选输出，标准化奖励后按相对优势更新。
  - 公式流程（文字说明）：给定源输入x，策略采样多个候选输出 $\hat{y}$；计算每个候选的复合奖励；在组内归一化得到优势；用优势更新策略，参考策略为初始SFT模型以防止漂移。

## 6. 主要结论与发现

1. **SG-SRL显著优于冷启动SFT**：在3-way排序中，SG-SRL在5000例中排名第一995次、第二2602次，而Cold-start SFT排名最后3185次；成对胜率SG-SRL占72.3%。
2. **语义RL作为中间训练而非最终优化**：直接优化语义奖励导致冗长奖励黑客，但中间检查点学习了有用的跨语言语义对齐。恢复阶段可保留语义增益，恢复流利性。
3. **跨领域迁移性**：SG-SRL在长文本翻译任务中提升BLEU（0.14→0.17）和语义维度（实体、事件、事实等），表明学到的是可重用的语义对齐。
4. **语言泛化到藏语**：使用域内编码器嵌入奖励（CINO）时，SG-SRL仍有效，两种参考语言（藏语或汉语）下BLEU相似，证明在无LLM重排序器时也可适应。
5. **语义-形式权衡的本质**：冗长奖励黑客源于奖励结构本身（相关性导向），而非特定RL算法（GRPO-Control同样失败）。约束太紧（RefLen）会压制语义学习。

## 8. 不足与局限

- **重排序器依赖性**：主实验中跨语言重排序器（如mGTE/Qwen3等）本身需要多语言覆盖，对于极低资源语言可能不存在或质量差，限制了直接适用性。
- **藏语实验的折扣**：藏语实验使用域内编码器奖励，但数据是平行描述而非新闻标题，任务、领域、奖励形式均与中泰实验不同，不能直接证明框架在新闻生成领域的泛化。
- **实验覆盖有限**：仅在一个模型（SmolLM3-3B）和一个语言对（中→泰）上进行主实验，未测试其他低资源语言（如印地语、斯瓦希里语）或其他规模模型（如7B/13B），通用性未证实。
- **评估偏差风险**：LLM judge（deepseek-v4-flash）本身在泰语和藏语上的能力未知，可能存在对某些输出风格的偏好，尤其是流利度和简洁性的判断可能受到训练数据影响。
- **计算资源未披露**：缺乏训练时间、GPU型号和数量、预算等信息，难以评估方法的实际成本。
- **长文本迁移实验样本量小**（100例），统计稳定性存疑。短文本翻译实验因区分度低未在主文报告，说明该方法在短输入上优势不明显。
- **恢复阶段复用同一个小平行语料**，可能引入过拟合风险，尽管实验未观察到明显退化，但理论上语义RL后的恢复可能压缩已经学习到的语义多样性。
