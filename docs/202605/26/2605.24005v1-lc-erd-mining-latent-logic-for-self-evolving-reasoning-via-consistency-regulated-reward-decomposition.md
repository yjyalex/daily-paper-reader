## 摘要
大语言模型推理能力的进化受到高质量过程数据稀缺的瓶颈制约。虽然通过内源奖励进行自对齐提供了一种解决方案，但挖掘有效监督面临三个挑战：(1) 通过模仿偏见的标签噪声，即奖励优先考虑统计似然而非逻辑真理，产生掩盖复合错误的“正确性幻觉”；(2) 粗粒度监督，即稀疏的全局结果（例如GRPO中）无法提供细粒度指导，将推理链视为整体；(3) 分布坍塌，即信号在未放大预训练偏见的情况下无法泛化。为解决这些问题，我们提出LC-ERD（逻辑一致内源奖励分解），一个将自对齐视为潜在结构挖掘的框架。我们通过聚合模型潜在逻辑专家（LLE）的共识推导出变分逻辑势，以去噪推理流形，并引入基于IGM原则的多智能体价值分解协议来量化单个步骤的效用。实验表明，LC-ERD提供了一条稳健的自进化路径，揭示了逻辑一致性与准确性之间的权衡，同时识别出标准奖励遗漏的高价值推理模式。我们的代码可在https://github.com/Reinhardmannn/LC-ERD获取。

## Abstract
The evolution of Large Language Model (LLM) reasoning is bottlenecked by the scarcity of high-quality process data. While self-alignment via endogenous rewards offers a solution, mining valid supervision faces three challenges: (1) Label Noise via Mimetic Bias, where rewards prioritize statistical likelihood over logical truth, creating a "correctness illusion" that masks compounding errors; (2) Coarse-Grained Supervision, where sparse global outcomes (e.g., in GRPO) fail to provide granular guidance, treating reasoning chains as monolithic; and (3) Distributional Collapse, where signals fail to generalize without amplifying pre-training biases. To address these, we introduce LC-ERD (Logic-Consistent Endogenous Reward Decomposition), a framework framing self-alignment as latent structure mining. We derive a Variational Logic Potential by aggregating consensus from the model's Latent Logic Expertise (LLE) to denoise the reasoning manifold, and introduce a Multi-Agent Value Decomposition protocol based on the IGM principle to quantify individual step utility. Experiments show LC-ERD delivers a robust self-evolution path, uncovering trade-offs between logic consistency and accuracy while identifying high-value reasoning patterns missed by standard rewards. Our code is available at https://github.com/Reinhardmannn/LC-ERD.

## 2. 方法论

### 核心思想
LC-ERD将推理过程建模为多智能体协作博弈，基于**IGM（Individual-Global-Max）原则**将全局奖励分解为每个步骤（相当于智能体）的细粒度价值，并通过**变分逻辑势（VLP）**对推理流形进行去噪，确保奖励反映逻辑正确性而非表面似然。

### 关键技术细节
1. **潜在逻辑专家（Latent Logic Expertise, LLE）发现**：
   - 对每个查询进行 $K$ 次独立采样，生成候选轨迹集合 $\mathcal{T}$；
   - 通过硬终端约束（如验证最终答案正确性）筛选出成功轨迹 $\mathcal{T}^*$；
   - 对每条成功轨迹计算置信权重 $\omega(\tau)$（基于联合概率），构建LLM的“逻辑专家分布” $\pi_{\text{LLE}}$：
     \[
     \pi_{\text{LLE}}(a_h|s_h) = \sigma\left( \frac{1}{\sum_k \omega(\tau^{(k)})} \sum_{\tau^{(k)} \in \mathcal{T}^*} \omega(\tau^{(k)}) \cdot \phi(s_h^{(k)}, a_h) \right)
     \]
     其中 $\sigma$ 是softmax，$\phi$ 是未归一化logits。此过程过滤掉随机幻觉，提取推理的“逻辑主干”。

2. **变分逻辑势（Variational Logic Potential, VLP）塑造**：
   - 定义势函数 $\Phi_{\text{LLE}}(s_h)$ 衡量当前策略 $\pi_\theta$ 与 LLE 分布的对齐程度：
     \[
     \Phi_{\text{LLE}}(s_h) = V(s_h) - \beta \cdot D_{\text{KL}}(\pi_\theta(\cdot|s_h) \parallel \pi_{\text{LLE}}(\cdot|s_h))
     \]
   - 势差 $\Delta\Phi_{\text{LLE}} = \gamma \Phi_{\text{LLE}}(s_{h+1}) - \Phi_{\text{LLE}}(s_h)$ 作为纠正信号。

3. **奖励分解**：
   - 模仿奖励 $\tilde{r}$ 由逆Soft贝尔曼算子从预训练模型logits中提取（反映统计似然）；
   - 最终LC-ERD奖励为两者之和：
     \[
     R_{\text{LC-ERD}}(s_h,a_h) = \tilde{r}(s_h,a_h) + \Delta\Phi_{\text{LLE}}(s_h,a_h)
     \]
   - 展开后得到：
     \[
     R_{\text{LC-ERD}} = \alpha \log \pi_{\text{base}}(a_h|s_h) + \beta \left[ D_{\text{KL}}(s_h) - \gamma D_{\text{KL}}(s_{h+1}) \right]
     \]
     第二项激励模型最小化每一步的逻辑散度。

4. **联合优化与信用分配**：
   - 全局价值 $Q_{\text{tot}}$ 分解为步骤级 $Q_h$（IGM原则）；
   - 总损失 $\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{TD}} + \lambda \sum_h \mathcal{D}_{\text{indiv}}$：
     - $\mathcal{L}_{\text{TD}}$：时序差分损失，稳定价值估计；
     - $\mathcal{D}_{\text{indiv}}$：个体一致性损失，惩罚当前策略logits与LLE分布的结构性偏差。

### 算法流程（文字说明）
1. 每轮训练迭代：从数据集采样一批查询；
2. **LLE发现**：对每个查询执行K次采样，筛选验证成功的轨迹，计算置信权重，构建LLE专家分布；
