## 摘要
多智能体辩论（MAD）提高了LLM智能体的准确性，但面临上下文快速增长的问题，限制了其在更大规模多智能体场景中的可扩展性。现有方法使用先验信号（如token级别的对数似然或LLM自我报告的置信度）来剪枝低效用通信。然而，这些信号在幻觉下变得不可靠，降低了依赖它们的MAD方法的准确性。我们提出SVR-MAD，一种受贝叶斯启发的MAD框架，将辩论前信号视为先验，辩论结果视为用于估计智能体正确性的后验式证据。SVR-MAD利用此证据逐步构建通信图，优先保留其答案经受住同伴挑战的智能体。在多个LLM和基准上的实验表明，与最准确的竞争MAD基线相比，SVR-MAD在匹配或提高准确性的同时，将token成本降低高达61%。

## Abstract
Multi-Agent Debate (MAD) improves LLM-agent accuracy but suffers from rapid context growth, limiting scalability in larger multi-agent settings. Existing methods prune low-utility communications using prior signals, such as token-level log-likelihoods or LLM self-reported confidence. However, these signals become unreliable under hallucination, degrading the accuracy of MAD methods that rely on them. We propose SVR-MAD, a Bayesian-inspired MAD framework that treats pre-debate signals as priors and debate outcomes as posterior-style evidence for estimating agent correctness. SVR-MAD uses this evidence to incrementally construct the communication graph, prioritizing agents whose answers survive peer challenges. Experiments across multiple LLMs and benchmarks show that SVR-MAD reduces token cost by up to 61% while matching or improving accuracy relative to the most accurate competing MAD baseline.

- **现有方法局限**：现有高效 MAD 方法依赖辩论前的先验信号（如 token 级对数似然 min-LL、困惑度 PPL、模型自报置信度）来裁剪低效用通信。然而这些先验信号在模型产生 **幻觉**（hallucination）时变得不可靠——错误智能体可能高度自信，正确智能体可能不确定——导致误判，降低最终准确率。
- **整体含义**：论文提出一种贝叶斯启发的框架 SVR-MAD，利用辩论过程中产生的 **后验证据**（posterior evidence）来动态构建通信图，优先保留那些能够经受同伴挑战的智能体，从而在降低通信成本的同时保持或提升准确率。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 核心思想
- 将辩论前信号视作 **先验**（prior），将辩论结果（智能体是否保留原答案）视为 **后验证据**（posterior evidence），用于估计智能体的正确性。
- 关键后验指标：**存活率（Survival Rate, SVR）** ——智能体在受到同伴挑战后保留其初始信念的频率。  
  公式：\( SVR = (r - c)/D \)，其中 \(D\) 为辩论总次数，\(r\) 为保留次数，\(c\) 为改变次数。

### 算法流程（Algorithm 1）
1. **初始化**：根据先验信号（min-LL/PPL/conf）为每个智能体 \(A_i\) 赋予初始正确性得分 \(Corr[A_i]\)。
2. **迭代构建通信图**（预算 \(B\) 递减）：
   - 选择得分最高的智能体作为接收者 \(A_r\)。
   - 从与 \(A_r\) 答案不同的智能体中，选出得分最高的 \(S\) 个作为挑战者。
   - 执行 \(S\) 对 \(挑战者 \to A_r\) 的成对辩论，根据 \(A_r\) 是否保留原答案更新其得分：  
     - 若 \(D>0\)，则 \(Corr[A_r] = SVR(D, r, c)\)（后验主导）；否则仍用先验。
   - 若 \(Corr[A_r] \geq \tau\)（阈值），提前终止，返回 \(A_r\) 的答案。
3. **后备机制**：若预算耗尽，使用多数投票（每个智能体以其观察到的后验答案做多数，平局由初始答案打破）。

### 关键参数
- \(S\)：每轮通信数（GPT-OSS: 2，DeepSeek: 3）
- \(\tau\)：接受阈值（由 \(SVR=100\%\) 且至少经过 \(C\) 个挑战者决定；GPT-OSS: \(C=2\)，DeepSeek: \(C=3\)）
- \(B_{max}\)：预算 = \(S \cdot (k + m)\)，\(k\) 为初始答案簇数，\(m\) 为最大簇大小。

## 3. 实验设计

### 数据集

### 对比方法
- **Self Consistency**：无辩论，直接多数投票。
- **GroupDebate**：随机子组内全上下文交流，组间仅传递最终答案。
- **SID-ET**：基于 token 对数似然（min-LL）阈值决定是否跳过辩论（超参数经调优选择 Pareto 最优设置）。
- **S²-MAD**：基于答案等价性/嵌入相似性裁剪通信链路。

### 指标
- **通信次数 (NComm)**：抽象图边数。
- **总 token 数 (Tok)**：具体开销。
- **准确率 (Acc)**。

- 6 个智能体/问题，最多 2 轮辩论，若 5 个智能体达成明确共识则提前终止。
- LLM：GPT-OSS-120B（文本-only）、DeepSeek-V3.1。采样参数：温度 1.0，top-p 0.95，top-k 40，max tokens 16384。

## 6. 主要结论与发现
- **成本-准确率权衡最优**：SVR-MAD 在全部四个设定下，匹配或超过所有 MAD 基线的准确率，同时将通信次数降低 48-75%、总 token 降低 38-61%（相比最准确基线）。
- **后验信号优于先验信号**：当问题变难（正确智能体少）时，先验信号（min-LL, PPL, conf）的正确率急剧下降（从 67-87% 降至 5-34%），而 SVR 仍保持 47-78%，表现远优于先验。
- **困难问题鲁棒性**：在只有 1-3 个初始正确智能体的难题上，SVR-MAD 正确数匹配或超过所有基线，且 token 成本更低（仅 S²-MAD 在 GPT-OSS/IMO 上多对一题正确，但花费 1.4-2.5 倍 token）。
- **消融验证**：SVR 作为后验信号在相同通信控制流程下，比使用 min-LL、PPL、conf 作为后验信号准确率高出最多 8.75 点（在匹配 token 成本时）。

## 7. 优点

### 方法创新
- **贝叶斯视角**：将辩论前信号与辩论后行为自然结合，先验引导初始选择，后验逐渐主导，解决了先验不可靠的问题。
- **后验信号 SVR 设计简洁有效**：基于“正确推理更可能坚持己见”的直觉，计算简单无需额外模型。
- **贪心增量构建通信图**：动态选择最有希望的智能体作为接收者，优先评估其与其他高阶挑战者的辩论，使得高质量答案能快速通过阈值，提前终止，降低成本。

### 实验设计亮点
- 对基线进行了公平调优（SID-ET 的 skip 率扫描，GroupDebate 选择最优分组）。
- 消融实验分离了 SVR 的作用，证明其在相同控制流程下优于其他后验信号。
- 分析了不同难度水平下信号的有效性，提供了深度洞察。
- 代码和额外细节开源（GitHub），增强可复现性。

## 8. 不足与局限
